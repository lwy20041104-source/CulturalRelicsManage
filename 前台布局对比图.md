## 🎨 样式细节

### 用户信息卡片

```css
.user-info-below {
  display: flex;           /* 弹性布局 */
  align-items: center;     /* 垂直居中 */
  gap: 8px;               /* 图标和文字间距 */
  color: #fdfbf7;         /* 文字颜色 */
  background: rgba(253, 251, 247, 0.15);  /* 半透明背景 */
  padding: 6px 12px;      /* 内边距 */
  border-radius: 6px;     /* 圆角 */
  font-size: 13px;        /* 字体大小 */
  border: 1px solid rgba(253, 251, 247, 0.25);  /* 边框 */
  margin-top: 8px;        /* 上边距 */
  width: fit-content;     /* 宽度自适应 */
}
```

**视觉效果**：
```
┌─────────────┐
│ 👤 张三     │  ← 半透明白色背景
└─────────────┘  ← 圆角边框
```

---

### 右上角按钮组

.header-right {

## 🎯 设计原则

### 1. 视觉层次

```
重要性排序：
1. 系统标题（最大、最醒目）
2. 操作按钮（右上角、易于点击）
3. 用户信息（次要信息）
4. 导航菜单（功能入口）
```

### 2. 空间利用

```
左侧：信息展示
- 系统标题
- 副标题
- 用户信息

右侧：操作按钮
- 通知
- 语言切换
- 退出登录
```

### 3. 用户体验

```
常见操作放在右上角：
✅ 符合用户习惯
✅ 易于点击
✅ 视觉清晰
```
