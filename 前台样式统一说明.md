### 2. 调整通知铃铛背景色

#### 问题
原来的通知铃铛使用白色背景，在前台的深色header中不协调。

#### 解决方案
添加专用样式，使用半透明白色背景适配深色背景。

**新增样式**：
```css
/* 通知铃铛样式调整 - 适配前台深色背景 */
.portal-notification :deep(.notification-button) {
  background: rgba(253, 251, 247, 0.15);
  border-color: rgba(253, 251, 247, 0.3);
  color: #fdfbf7;
}

.portal-notification :deep(.notification-button:hover) {
  background: rgba(253, 251, 247, 0.25);
  border-color: rgba(253, 251, 247, 0.4);
  color: #fdfbf7;
}
```

**效果**：
- 半透明白色背景（15%透明度）
- 悬停时背景加深（25%透明度）
- 文字和图标为米白色
- 与深色背景完美融合

## 🔄 语言切换逻辑

### 兼容性处理

前台页面使用简化的语言代码（'zh'/'en'），而LanguageSwitcher组件使用完整的语言代码（'zh-CN'/'en-US'）。

**解决方案**：添加映射逻辑

```javascript
// 读取localStorage，兼容两种格式
const storedLocale = localStorage.getItem('locale') || localStorage.getItem('portalLocale') || 'zh-CN'

// 映射为前台使用的简化格式
const locale = ref(storedLocale.startsWith('zh') ? 'zh' : 'en')

// 监听locale变化，同步到localStorage
watch(locale, (newVal) => {
  const fullLocale = newVal === 'zh' ? 'zh-CN' : 'en-US'
  localStorage.setItem('locale', fullLocale)
  localStorage.setItem('portalLocale', newVal)
})

// 监听localStorage的locale变化（由LanguageSwitcher触发）
const syncLocale = () => {
  const storedLocale = localStorage.getItem('locale')
  if (storedLocale) {
    locale.value = storedLocale.startsWith('zh') ? 'zh' : 'en'
  }
}
```

**工作流程**：
1. LanguageSwitcher组件切换语言（zh-CN/en-US）
2. 保存到localStorage的'locale'键
3. syncLocale函数检测变化
4. 映射为前台格式（zh/en）
5. 更新前台页面显示
