# 通知图标样式统一说明

## 📅 修改日期
2026年4月24日

---

## 🎯 修改目标

将消息通知图标的背景色调整为与中英文切换按钮完全一致的样式。

---

## 🎨 样式对比

### 修改前

**通知铃铛**：
```css
.notification-button {
  background: rgba(253, 251, 247, 0.15);
  border-color: rgba(253, 251, 247, 0.3);
  color: #fdfbf7;
  /* 缺少 padding 和 border-radius */
}
```

**语言切换器**：
```css
.language-switcher {
  background: rgba(253, 251, 247, 0.15);
  border: 1px solid rgba(253, 251, 247, 0.3);
  color: #fdfbf7;
  padding: 8px 12px;
  border-radius: 6px;
}
```

**问题**：
- ❌ 通知铃铛缺少内边距
- ❌ 通知铃铛缺少圆角
- ❌ 样式不完全一致

---

### 修改后

**通知铃铛**：
```css
.portal-notification :deep(.notification-button) {
  background: rgba(253, 251, 247, 0.15);
  border: 1px solid rgba(253, 251, 247, 0.3);
  color: #fdfbf7;
  padding: 8px 12px;           /* 新增 */
  border-radius: 6px;          /* 新增 */
  transition: all 0.3s;        /* 新增 */
}
```

**语言切换器**：
```css
.portal-language :deep(.language-switcher) {
  background: rgba(253, 251, 247, 0.15);
  border: 1px solid rgba(253, 251, 247, 0.3);
  color: #fdfbf7;
  padding: 8px 12px;
  border-radius: 6px;
  transition: all 0.3s;
}
```

**优势**：
- ✅ 样式完全一致
- ✅ 内边距相同（8px 12px）
- ✅ 圆角相同（6px）
- ✅ 过渡效果相同（0.3s）

## 🔍 详细样式规格

### 共同样式属性

| 属性 | 值 | 说明 |
|------|-----|------|
| background | rgba(253, 251, 247, 0.15) | 15%透明度的米白色 |
| border | 1px solid rgba(253, 251, 247, 0.3) | 30%透明度的米白色边框 |
| color | #fdfbf7 | 米白色文字 |
| padding | 8px 12px | 上下8px，左右12px |
| border-radius | 6px | 圆角6px |
| transition | all 0.3s | 所有属性0.3秒过渡 |

| 属性 | 值 | 说明 |

| color | #fdfbf7 | 米白色文字（保持不变） |
