# 测试指南

## 快速测试步骤

### 1. 启动系统

```bash

cd frontend
npm run dev

# 终端2 - 启动后端（如果未启动）
cd backend
mvn spring-boot:run
```

## 常见问题

### Q: 页面显示空白
**A**: 
1. 检查浏览器控制台是否有错误
2. 检查后端服务是否启动
3. 刷新页面重试

### Q: 数据不显示
**A**:
1. 打开浏览器开发者工具
2. 查看Network标签，检查API请求是否成功
3. 检查后端日志

### Q: 保存失败
**A**:
1. 检查表单是否填写完整
2. 检查密码是否符合要求（6-20位，包含数字和字母）
3. 检查手机号格式是否正确
4. 检查邮箱格式是否正确

### Q: 借展人的博物馆不显示
**A**:
1. 检查该借展人是否已关联博物馆
2. 检查user_museum表中是否有对应记录
3. 刷新页面重试

### Q: 删除博物馆后还能看到
**A**:
- 博物馆删除是逻辑删除，状态会变为"无合作"
- 如果需要完全隐藏，可以筛选"有合作"状态
