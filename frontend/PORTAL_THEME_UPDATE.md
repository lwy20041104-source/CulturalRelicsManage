# 前台门户主题功能更新

## 更新内容

已为前台门户系统（PublicPortalView）添加主题切换功能，与后台管理系统保持一致。

## 实现的功能

### ✅ 主题切换器集成

**位置**：顶部导航栏右侧
- 通知铃铛 🔔
- **主题切换器 🎨** ← 新增
- 语言切换器 🌐
- 退出按钮

### ✅ 样式适配

已将前台门户的关键样式更新为使用CSS变量：

1. **顶部导航栏**
   - 背景色：`var(--bg-aside)`
   - 文字颜色：`var(--text-primary)`
   - 边框颜色：`var(--border-light)`

2. **导航菜单**
   - 默认背景：`var(--bg-main)`
   - 悬停背景：`var(--bg-hover)`
   - 激活背景：`var(--color-primary)`
   - 边框颜色：`var(--border-light)`

3. **用户信息**
   - 用户名颜色：`var(--color-primary-light)`
   - 悬停颜色：`var(--color-primary)`

4. **页面背景**
   - 渐变背景：`var(--bg-aside)` → `var(--bg-main)`

## 使用方法

### 前台用户切换主题

1. 登录前台门户系统
2. 点击顶部导航栏的画笔图标 🎨
3. 在下拉菜单中选择喜欢的主题：
   - 🏛️ 古典棕色
   - 🏺 青瓷雅韵
   - 🎨 青花瓷韵
   - 🪵 紫檀沉香
   - 💎 墨玉清幽
   - ✨ 琥珀金辉
4. 主题立即生效并自动保存

### 主题同步

- 前台和后台的主题选择是**独立的**
- 每个系统可以使用不同的主题
- 主题偏好保存在浏览器本地存储中

```css
/* 使用CSS变量 */

  color: #ffffff;
```
