### 5. 单条操作测试

- [ ] **标记已读**
  - 选择一条未读通知
  - 点击"标记已读"按钮
  - 通知状态变为"已读"
  - 标题不再加粗
  - 显示成功提示

- [ ] **删除通知**
  - 选择一条通知
  - 点击"删除"按钮
  - 弹出确认对话框
  - 点击"确定"
  - 通知被删除
  - 显示成功提示

- [ ] **取消删除**
  - 选择一条通知
  - 点击"删除"按钮
  - 弹出确认对话框
  - 点击"取消"
  - 通知保留
  - 无任何提示

### 6. 批量操作测试

- [ ] **选择通知**
  - 勾选表格第一列的复选框
  - 选中的通知高亮显示
  - 顶部显示"已选择 X 条"

- [ ] **全选功能**
  - 点击表头的复选框
  - 当前页所有通知被选中
  - 再次点击取消全选

- [ ] **批量标记已读**
  - 选中多条未读通知
  - 点击顶部"批量标记已读"按钮
  - 所有选中通知变为已读
  - 显示成功提示
  - 选择状态自动清空

- [ ] **批量删除**
  - 选中多条通知
  - 点击顶部"批量删除"按钮
  - 弹出确认对话框，显示删除数量
  - 点击"确定"
  - 所有选中通知被删除
  - 显示成功提示
  - 选择状态自动清空

- [ ] **批量删除取消**
  - 选中多条通知
  - 点击顶部"批量删除"按钮
  - 弹出确认对话框
  - 点击"取消"
  - 通知保留
  - 选择状态保持
