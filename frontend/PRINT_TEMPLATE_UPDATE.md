# 打印模板更新说明

## 更新内容

打印模板已经完全重新设计，现在只显示档案信息和打印时间，无菜单栏等多余元素。

## 新打印模板特点

### 1. 简洁设计
- ✅ 只包含档案信息和打印时间
- ✅ 无菜单栏、按钮等UI元素
- ✅ 使用原生HTML表格布局
- ✅ 专业的文档格式

### 2. 打印内容
- 档案详情标题（居中）
- 打印时间（自动生成，格式：YYYY-MM-DD HH:mm:ss）
- 档案基本信息表格
- 关联文物信息表格
- 档案描述
- 档案管理信息表格
- 附件数量

### 3. 布局优化
- A4纸张适配
- 20mm页边距
- 表格边框清晰
- 避免内容跨页断开

## 如何查看新模板

### 方法1：硬刷新浏览器（推荐）
1. 在浏览器中按以下快捷键：
   - **Windows**: `Ctrl + Shift + R` 或 `Ctrl + F5`
   - **Mac**: `Cmd + Shift + R`
2. 这将清除缓存并重新加载页面

### 方法2：清除浏览器缓存
1. 打开浏览器开发者工具（F12）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 方法3：无痕模式测试
1. 打开浏览器的无痕/隐私模式
2. 访问系统
3. 测试打印功能

### 方法4：重启开发服务器
```bash
# 停止当前服务器（Ctrl + C）
# 然后重新启动
cd frontend
npm run dev
```

## 验证方法

1. 登录系统
2. 进入"档案管理"页面
3. 点击任意档案的"更多" > "打印"
4. 查看打印预览对话框
5. 应该看到：
   - 顶部：档案详情标题 + 打印时间
   - 中间：表格形式的档案信息

6. 点击"打印"按钮
7. 在浏览器打印预览中应该只看到档案信息，无菜单栏

## 技术细节

### 模板结构
```html
<div id="printArea">
  <!-- 打印标题 -->
  <div class="print-header">
    <h1>档案详情</h1>
    <div class="print-time">打印时间: 2026-04-27 14:30:00</div>
  </div>

  <!-- 档案基本信息 -->
  <div class="info-section">
    <h3>档案基本信息</h3>
    <table class="info-table">
      <tr>
        <td class="label">档案编号:</td>
        <td class="value">AR20260427001</td>
        <td class="label">档案类型:</td>
        <td class="value">完整档案</td>
      </tr>
      ...
    </table>
  </div>
  
  <!-- 其他信息区块 -->
  ...
</div>
```

### 打印样式
```css
@media print {
  /* 隐藏所有页面元素 */
  body * {
    visibility: hidden;
  }

  /* 只显示打印区域 */
  #printArea,
  #printArea * {
    visibility: visible;
  }

  /* 打印区域占满页面 */
  #printArea {
    position: absolute;
    left: 0;
    top: 0;
    width: 100%;
    padding: 20mm;
  }
}
```

## 常见问题

### Q1: 为什么我看不到新的打印模板？
**A**: 这是浏览器缓存问题。请使用上面的"硬刷新"方法清除缓存。

### Q2: 打印预览中还是显示菜单栏？
**A**: 请确保：
1. 已经硬刷新浏览器
2. 使用的是最新代码
3. 开发服务器已重启

### Q3: 打印时间没有显示？
**A**: 请检查：
1. 翻译文件是否包含 `archive.printTime` 键
2. `currentPrintTime` 变量是否正确设置
3. 浏览器控制台是否有错误

### Q4: 表格边框在打印时不显示？
**A**: 这是CSS问题，已在打印样式中使用 `!important` 强制显示边框。
