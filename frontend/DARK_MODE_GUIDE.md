# 暗黑模式功能说明

## 功能概述

为系统添加了完整的暗黑模式支持，每个主题都有对应的暗黑版本。用户可以在浅色模式和暗黑模式之间自由切换，提供更舒适的夜间使用体验。

## 主要特性

### ✅ 完整的暗黑模式支持

**6个主题 × 2种模式 = 12种视觉风格**

每个主题都精心设计了暗黑版本：
- 🏛️ 古典棕色 - 浅色/暗黑
- 🏺 青瓷雅韵 - 浅色/暗黑
- 🎨 青花瓷韵 - 浅色/暗黑
- 🪵 紫檀沉香 - 浅色/暗黑
- 💎 墨玉清幽 - 浅色/暗黑
- ✨ 琥珀金辉 - 浅色/暗黑

### ✅ 暗黑模式切换器

**位置**：顶部导航栏
- 通知铃铛 🔔
- **暗黑模式切换 🌙/☀️** ← 新增
- 主题切换器 🎨
- 语言切换器 🌐
- 退出按钮

**图标**：
- 浅色模式：显示月亮图标 🌙
- 暗黑模式：显示太阳图标 ☀️

### ✅ 智能颜色适配

暗黑模式下的颜色设计原则：
1. **背景色**：深色调，降低亮度
2. **文字颜色**：浅色调，提高对比度
3. **主色调**：适当提亮，保持识别度
4. **边框颜色**：柔和过渡，不刺眼

## 使用方法

### 切换暗黑模式

1. **后台管理系统**
   - 点击顶部导航栏的月亮/太阳图标
   - 系统立即切换到暗黑/浅色模式
   - 显示切换成功提示

2. **前台门户系统**
   - 同样点击月亮/太阳图标
   - 立即切换模式

### 组合使用

暗黑模式与主题切换独立工作：
1. 先选择喜欢的主题（如青瓷雅韵）
2. 再切换到暗黑模式
3. 得到"青瓷雅韵 - 暗黑版"

### 持久化存储

- 暗黑模式选择自动保存
- 主题选择自动保存
- 下次访问自动应用上次的设置

## 技术实现

### 1. 主题系统扩展

每个主题现在包含两套颜色配置：

```javascript
classic: {
  name: '古典棕色',
  icon: '🏛️',
  colors: {
    // 浅色模式颜色
    primary: '#a67c52',
    bgMain: '#f5ede0',
    textPrimary: '#3d2f1f',
    // ...
  },
  darkColors: {
    // 暗黑模式颜色
    primary: '#c9a875',
    bgMain: '#1a1612',
    textPrimary: '#e8e0d5',
    // ...
  }
}
```

### 2. 暗黑模式状态管理

```javascript
// 暗黑模式状态
const isDarkMode = ref(localStorage.getItem('darkMode') === 'true')

// 切换暗黑模式
export const toggleDarkMode = () => {
  const newDarkMode = !isDarkMode.value
  applyTheme(currentTheme.value, newDarkMode)
}

// 应用主题时选择颜色集
const applyTheme = (themeName, dark = isDarkMode.value) => {
  const theme = themes[themeName]
  const colors = dark ? theme.darkColors : theme.colors
  
  // 设置CSS变量
  root.style.setProperty('--color-primary', colors.primary)
  // ...
}
```

### 3. 暗黑模式切换器组件

```vue
<template>
  <el-button class="dark-mode-toggle" circle @click="handleToggle">
    <el-icon>
      <Moon v-if="!isDarkMode" />
      <Sunny v-else />
    </el-icon>
  </el-button>
</template>

<script setup>
import { useTheme } from '../composables/useTheme'

const { isDarkMode, toggleDarkMode } = useTheme()

const handleToggle = () => {
  toggleDarkMode()
  // 显示成功提示
}
</script>
```

### 4. CSS类名支持

暗黑模式激活时，根元素会添加 `dark-mode` 类：

```javascript
if (dark) {
  root.classList.add('dark-mode')
} else {
  root.classList.remove('dark-mode')
}
```

可以使用此类名添加额外的暗黑模式样式：

```css
.dark-mode .some-element {
  /* 暗黑模式特定样式 */
}
```

## 护眼设计

### 暗黑模式的优势

1. **减少眼睛疲劳**
   - 降低屏幕亮度
   - 减少蓝光刺激
   - 适合长时间使用

2. **节省电量**
   - OLED屏幕显示黑色时更省电
   - 延长设备续航时间

3. **夜间友好**
   - 不刺眼
   - 不影响睡眠
   - 适合夜间工作

### 对比度优化

暗黑模式下的文字对比度：
- 主要文字：≥ 7:1（AAA级）
- 次要文字：≥ 4.5:1（AA级）
- 符合WCAG 2.1标准

## 常见问题

### Q: 暗黑模式会影响性能吗？
A: 不会，暗黑模式使用CSS变量，性能影响可忽略不计。

### Q: 可以自动切换暗黑模式吗？
A: 当前版本需要手动切换，自动切换功能将在未来版本推出。

### Q: 暗黑模式下图片会变暗吗？
A: 图片不会自动变暗，保持原始显示效果。

### Q: 如何恢复浅色模式？
A: 再次点击切换按钮即可恢复浅色模式。

### Q: 暗黑模式和主题切换冲突吗？
A: 不冲突，它们是独立的功能，可以组合使用。
