- **技术栈与构建**：采用 Vue 3 (Composition API) + Vite 构建，使用 Pinia 进行状态管理，Vue Router 处理路由，Element Plus 作为 UI 组件库。
- **目录结构**：`src/api` 封装 Axios 请求及业务接口；`src/views` 区分后台管理视图（如 `RelicsView`）与公众门户视图（如 `PublicPortalView`）；`src/components` 存放通用组件（如 `Relic3DViewer`）；`src/composables` 提供主题切换等逻辑复用。
- **路由与权限**：`router/index.js` 实现基于角色的访问控制（RBAC），通过 `sessionStorage` 存储 Token 与权限标识，在路由守卫中拦截未授权访问并区分后台管理员与前台借用人角色。
- **网络通信**：`api/request.js` 统一处理 HTTP 请求拦截与业务错误码解析；`utils/websocket.js` 基于 SockJS/STOMP 实现实时通知推送。
- **特色功能**：`composables/useTheme.js` 实现包含 6 种预设配色及暗黑模式的动态主题系统；`components/Relic3DViewer.vue` 集成 Three.js 与 Sketchfab 实现文物 3D 模型交互展示。
