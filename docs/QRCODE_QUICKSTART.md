# 文物二维码功能 - 快速开始

## 5分钟快速上手

### 1. 生成二维码（30秒）

1. 登录系统，进入"文物管理"页面
2. 在文物列表中找到任意文物
3. 点击操作列的"二维码"按钮
4. 二维码自动生成并显示

### 2. 下载二维码（10秒）

1. 在二维码对话框中
2. 点击"下载二维码"按钮
3. 图片自动保存到下载文件夹
4. 文件名：`二维码_{文物编号}_{文物名称}.png`

### 3. 打印二维码（20秒）

1. 在二维码对话框中
2. 点击"打印二维码"按钮
3. 打印预览窗口自动打开
4. 点击"打印"按钮完成打印

### 4. 扫描查看（10秒）

1. 使用手机扫描二维码
2. 自动跳转到文物信息页面
3. 无需登录即可查看基本信息

## 常用操作

### 查看二维码
```
文物列表 → 点击"二维码"按钮 → 查看二维码
```

### 下载二维码
```
二维码对话框 → 点击"下载二维码" → 保存到本地
```

### 打印二维码
```
二维码对话框 → 点击"打印二维码" → 打印预览 → 打印
```

### 扫描二维码
```
手机扫码 → 打开链接 → 查看文物信息
```

## 技术要点

### 后端
- **依赖**：Google ZXing 3.5.1
- **工具类**：`QRCodeUtil.java`
- **接口**：`GET /relics/{id}/qrcode`

### 前端
- **页面**：`RelicsView.vue`（生成）、`QRCodeScanView.vue`（扫描）
- **路由**：`/qrcode/:id`
- **API**：`generateQRCodeApi(id, baseUrl)`

## 应用场景

1. 扫描二维码

3. 查看存储信息

## 注意事项

1. **网络访问**：扫描页面需要网络连接
2. **URL配置**：确保baseUrl配置正确
3. **图片质量**：二维码默认200x200像素
4. **打印设置**：建议使用A4纸张

## 故障排除

### 问题：二维码生成失败
**解决**：检查后端服务是否运行，ZXing库是否正确引入

### 问题：扫描后无法访问
**解决**：确认前端服务运行，检查网络连接

### 问题：文物信息不显示
**解决**：检查API接口是否正常，文物ID是否正确
