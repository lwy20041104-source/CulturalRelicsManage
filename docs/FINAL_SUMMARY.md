## 三、项目架构总结

### 前后台分离架构

```
┌─────────────────────────────────────┐
│         前台系统（借展人端）          │
│  - 登录：/portal-login               │
│  - 访客：/portal-guest（未登录）     │
│  - 门户：/portal（已登录）           │
│  - 主题：独立存储                    │
│  - 风格：古典棕色                    │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│         后台系统（管理员端）          │
│  - 登录：/login                      │
│  - 首页：/dashboard                  │
│  - 主题：独立存储                    │
│  - 风格：可自定义                    │
└─────────────────────────────────────┘
```

### 用户角色体系

```
┌─────────────────────────────────────┐
│         后台角色（3种）              │
│  - ADMIN：系统管理员（全部权限）     │
│  - CURATOR：文物保管员（文物管理）   │
│  - APPROVER：借展审批员（审批权限）  │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│         前台角色（1种）              │
│  - LOANER：借展人（浏览、申请）      │
│  - 必须关联博物馆                    │
└─────────────────────────────────────┘
```

### 主题系统

```
6个预设主题 × 2种模式 = 12种视觉风格

主题：
1. 古典棕色（默认）
2. 青瓷雅韵
3. 青花瓷韵
4. 紫檀沉香
5. 墨玉清幽
6. 琥珀金辉

模式：
- 浅色模式（默认）
- 暗黑模式（护眼）
```

## 四、技术栈总结

### 后端技术
- **框架**: Spring Boot 2.7.14
- **安全**: Spring Security + JWT
- **持久层**: MyBatis-Plus 3.5.3.1
- **数据库**: MySQL 8.0 + Redis
- **工具库**: Lombok、Hutool、Apache POI、iText、Jsoup
- **通信**: WebSocket（STOMP协议）
- **邮件**: Spring Boot Mail
- **短信**: 阿里云SMS SDK

### 前端技术
- **框架**: Vue 3.3 + Vite 4
- **UI**: Element Plus
- **图表**: ECharts 5
- **HTTP**: Axios
- **国际化**: Vue I18n
- **WebSocket**: SockJS Client + @stomp/stompjs
- **路由**: Vue Router 4
