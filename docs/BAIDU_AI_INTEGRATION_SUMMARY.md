##  核心功能

### 1. 混合识别策略

\\\
用户上传图片
    
检查百度AI是否可用
    
是  调用百度AI识别
    
检查置信度
    
高置信度(60%)  直接返回AI结果
    
低置信度(<60%)  结合规则识别
    
返回最终结果

否  使用规则识别
    
返回规则识别结果
\\\

##  使用方法

### 1. 申请百度AI密钥

1. 访问：https://ai.baidu.com/
2. 注册并登录
3. 创建应用（选择"图像识别"）
4. 获取APP_ID、API_KEY、SECRET_KEY

### 2. 配置密钥

编辑ackend/src/main/resources/application.yml：

\\\yaml
baidu:
  ai:
    enabled: true  # 启用百度AI
    app-id: 你的APP_ID
    api-key: 你的API_KEY
    secret-key: 你的SECRET_KEY
    confidence-threshold: 0.6
\\\

### 3. 重启后端服务

\\\ash
cd backend
mvn clean install
mvn spring-boot:run
\\\

### 4. 测试功能

1. 登录系统
2. 进入文物管理
3. 添加/编辑文物
4. 上传图片
5. 点击"AI识别"
6. 查看识别结果
