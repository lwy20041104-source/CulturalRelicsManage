# 文物二维码局域网访问快速指南

## 📱 问题：其他设备能扫描二维码吗？

**简短回答**：
- ❌ 如果只在本机运行（localhost），其他设备**无法访问**
- ✅ 如果配置局域网访问，同一WiFi下的设备**可以访问**
- ✅ 如果部署到服务器，任何设备**都可以访问**

## 🚀 快速配置（5分钟搞定）

### 步骤1：获取你的电脑IP地址

**Windows系统**：
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入 `ipconfig`，回车
3. 找到 `IPv4 地址`，例如：`192.168.1.100`

**Mac系统**：
1. 打开终端
2. 输入 `ifconfig | grep "inet "`
3. 找到类似 `192.168.1.100` 的地址

### 步骤2：确认配置（已完成✅）

前端和后端已经配置好了自动检测功能，无需修改代码！

**前端配置**（`frontend/vite.config.js`）：
```javascript
server: {
  host: '0.0.0.0'  // ✅ 已配置
}
```

**API配置**（`frontend/src/api/request.js`）：
```javascript
// ✅ 已自动检测环境
const getBaseURL = () => {
  const hostname = window.location.hostname
  if (hostname === 'localhost') {
    return 'http://localhost:8080/api'
  }
  return `${window.location.protocol}//${hostname}:8080/api`
}
```

### 步骤3：启动服务

```bash
# 启动后端（在backend目录）
mvn spring-boot:run

# 启动前端（在frontend目录）
npm run dev
```

### 步骤4：在手机上访问

1. **确保手机和电脑连接同一WiFi**
2. 在手机浏览器输入：`http://你的IP地址:5173`
   - 例如：`http://192.168.1.100:5173`
3. 如果能看到登录页面，说明配置成功！

### 步骤5：测试二维码

1. 在手机浏览器访问：`http://192.168.1.100:5173/public-relics`
2. 点击任意文物查看详情
3. 查看二维码（应该显示你的IP地址）
4. 用另一台手机扫描二维码
5. 应该能看到文物信息！

## 🔧 可能遇到的问题

### 问题1：手机无法访问电脑IP

**原因**：Windows防火墙阻止了访问

**解决方法**：

#### 方法A：临时关闭防火墙（测试用）
1. 打开 `控制面板` → `Windows Defender 防火墙`
2. 点击 `启用或关闭 Windows Defender 防火墙`
3. 选择 `关闭 Windows Defender 防火墙`（专用网络）
4. 测试完成后记得重新开启

#### 方法B：添加防火墙规则（推荐）
1. 打开 `控制面板` → `Windows Defender 防火墙` → `高级设置`
2. 点击 `入站规则` → `新建规则`
3. 选择 `端口` → 下一步
4. 选择 `TCP`，输入端口 `5173,8080` → 下一步
5. 选择 `允许连接` → 下一步
6. 全部勾选 → 下一步
7. 输入名称 `文物管理系统` → 完成

### 问题2：二维码显示localhost

**原因**：使用localhost访问生成的二维码

**解决方法**：
- ❌ 不要用 `http://localhost:5173` 访问
- ✅ 要用 `http://192.168.1.100:5173` 访问（你的IP）
- 这样生成的二维码才会包含IP地址

### 问题3：后端API请求失败

**原因**：后端没有绑定到所有网络接口

**解决方法**：

修改 `backend/src/main/resources/application.yml`：
```yaml
server:
  port: 8080
  address: 0.0.0.0  # 添加这一行
  servlet:
    context-path: /api
```

重启后端服务。

### 问题4：IP地址经常变化

**原因**：路由器动态分配IP

**解决方法**：

#### 在路由器中设置静态IP
1. 登录路由器管理页面（通常是 `192.168.1.1` 或 `192.168.0.1`）
2. 找到 `DHCP设置` 或 `静态IP分配`
3. 绑定你的电脑MAC地址到固定IP
4. 保存并重启路由器
