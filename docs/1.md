Design and Implementation of Museum Cultural Relics Digital Management System

***\*摘  要\****

 ***摘  要***

本文设计并实现了一个基于Spring Boot和Vue 3的博物馆文物数字化管理系统。该系统采用前后端分离架构，实现了文物信息管理、3D文物展示、借展管理、修复管理、数字化档案管理、AI智能查询等27个核心功能模块，覆盖文物管理的全生命周期。

系统后端采用Spring Boot框架，集成Spring Security实现JWT无状态认证和基于角色的访问控制（RBAC），使用MyBatis作为持久层框架，MySQL作为主数据库，Redis用于缓存和登录安全控制。前端采用Vue 3 Composition API，使用Element Plus组件库构建用户界面，集成Three.js实现3D文物展示，使用ECharts进行数据可视化。系统实现了WebSocket实时通信，支持消息实时推送；集成百度AI SDK实现图像识别功能；采用Google ZXing生成二维码标签；支持Excel、PDF、Word等多种格式的数据导入导出。

系统核心算法包括：BCrypt密码加密算法、JWT认证机制、AI智能查询相关度评分算法、图片智能提取算法、基于内容的协同过滤推荐算法等。数据库设计包含20+张表，采用索引优化、外键约束、软删除等设计策略，确保数据完整性和查询性能。

经过功能测试、性能测试和安全测试，系统运行稳定，功能完善，用户体验良好，能够满足博物馆文物数字化管理的实际需求，为文物保护和展示提供了有效的技术支持。

关键词：文物管理；数字化；Spring Boot；Vue 3；3D展示

***Abstract***

This paper designs and implements a museum cultural relics digital management system based on Spring Boot and Vue 3. The system adopts a front-end and back-end separation architecture, implementing 27 core functional modules including cultural relic information management, 3D relic display, loan management, repair management, digital archive management, and AI intelligent query, covering the entire lifecycle of cultural relic management.

The backend adopts the Spring Boot framework, integrates Spring Security to implement JWT stateless authentication and role-based access control (RBAC), uses MyBatis as the persistence layer framework, MySQL as the main database, and Redis for caching and login security control. The frontend adopts Vue 3 Composition API, uses Element Plus component library to build user interface, integrates Three.js to implement 3D relic display, and uses ECharts for data visualization. The system implements WebSocket real-time communication to support real-time message push; integrates Baidu AI SDK to implement image recognition function; uses Google ZXing to generate QR code labels; supports data import and export in multiple formats such as Excel, PDF, and Word.

The core algorithms of the system include: BCrypt password encryption algorithm, JWT authentication mechanism, AI intelligent query relevance scoring algorithm, intelligent image extraction algorithm, content-based collaborative filtering recommendation algorithm, etc. The database design contains 20+ tables, adopting index optimization, foreign key constraints, soft deletion and other design strategies to ensure data integrity and query performance.

After functional testing, performance testing and security testing, the system runs stably, has complete functions, and has good user experience. It can meet the actual needs of museum cultural relics digital management and provide effective technical support for cultural relic protection and display.

***Key words:*** cultural relic management; digitalization; Spring Boot; Vue 3; 3D display

## 1.1 研究背景及意义

### 1.1.1 研究背景

随着信息技术的快速发展和文化遗产保护意识的不断提高，博物馆作为文化遗产保护和展示的重要机构，面临着数字化转型的迫切需求。传统的文物管理方式主要依赖纸质档案和人工记录，存在信息检索困难、数据易丢失、管理效率低下等问题。同时，随着博物馆藏品数量的不断增加，借展、修复、维护等业务流程日益复杂，传统管理模式已难以满足现代博物馆管理的需求。

近年来，国家高度重视文化遗产的数字化保护工作。2020年，国家文物局发布《关于推进博物馆改革发展的指导意见》，明确提出要"推进智慧博物馆建设，加强文物数字化保护利用"。2021年，《"十四五"文物保护和科技创新规划》进一步强调要"加快推进文物资源数字化转化和开发"。这些政策为博物馆文物数字化管理系统的研发提供了政策支持和发展方向。

在技术层面，云计算、大数据、人工智能、3D建模等新兴技术的成熟应用，为文物数字化管理提供了技术支撑。特别是Web3D技术的发展，使得文物的三维展示成为可能，为公众提供了更加直观、生动的文物观赏体验。同时，移动互联网的普及也为文物信息的传播和共享创造了条件。

基于以上背景，开发一套功能完善、技术先进的博物馆文物数字化管理系统，对于提高文物管理效率、保护文化遗产、促进文化传播具有重要意义。

### 1.1.2 研究意义

本研究的意义主要体现在以下几个方面：

**（1）理论意义**

本研究将现代信息技术与文物管理业务深度融合，探索了文物数字化管理的新模式。通过系统化的需求分析、架构设计和功能实现，为博物馆信息化建设提供了理论参考和实践案例。同时，本研究在AI智能查询、3D文物展示、实时通信等方面进行了技术创新，丰富了文物数字化管理的理论体系。

**（2）实践意义**

① **提高管理效率**：系统实现了文物信息的集中管理和快速检索，大幅提高了文物管理的效率。通过自动化的业务流程（如借展审批、修复申请等），减少了人工操作，降低了管理成本。

② **保障数据安全**：采用多层次的安全机制（JWT认证、角色权限控制、数据加密等），确保文物信息的安全性和完整性。同时，系统支持数据备份和恢复，有效防止数据丢失。

③ **促进文化传播**：通过3D文物展示、二维码标签、AI智能查询等功能，为公众提供了更加便捷、生动的文物观赏和学习方式，促进了文化遗产的传播和普及。

④ **支持科学决策**：系统提供了丰富的统计分析功能，通过数据可视化展示文物的分类分布、状态统计、借展趋势等信息，为博物馆管理者提供决策支持。

⑤ **推动行业发展**：本系统采用主流的技术栈和开放的架构设计，具有良好的可扩展性和可维护性，可为其他博物馆的信息化建设提供参考和借鉴。

**（3）社会意义**

文物是民族文化的重要载体，承载着历史记忆和文化传承。通过数字化手段对文物进行保护和展示，不仅有助于文物的长期保存，也为公众了解历史、传承文化提供了新的途径。本系统的研发和应用，对于提高全民文化素养、增强文化自信具有积极的社会意义。

## 1.2 国内外的发展状况及应用

### 1.2.1 国内外发展状况

**（1）国外发展状况**

国外博物馆的数字化建设起步较早，技术应用较为成熟。美国史密森尼博物馆（Smithsonian Institution）早在20世纪90年代就开始建设数字化管理系统，目前已实现了藏品的全面数字化管理和在线展示。英国大英博物馆（British Museum）开发了先进的藏品管理系统（Collections Management System），支持多语言、多媒体的文物信息管理。法国卢浮宫博物馆（Louvre Museum）推出了虚拟参观系统，利用VR/AR技术为观众提供沉浸式的观展体验。

在技术应用方面，国外博物馆普遍采用了以下先进技术：
- **3D扫描与建模技术**：利用激光扫描、摄影测量等技术对文物进行三维数字化采集
- **人工智能技术**：应用图像识别、自然语言处理等AI技术实现文物的智能分类和检索
- **区块链技术**：利用区块链技术确保文物信息的真实性和不可篡改性
- **云计算技术**：采用云存储和云计算平台实现文物数据的集中管理和共享

**（2）国内发展状况**

我国博物馆数字化建设起步相对较晚，但发展迅速。故宫博物院是国内博物馆数字化建设的典范，开发了"数字故宫"平台，实现了文物的数字化管理、在线展示和虚拟参观。中国国家博物馆建设了"智慧国博"系统，集成了藏品管理、展览管理、观众服务等多个子系统。上海博物馆、南京博物院等大型博物馆也相继建设了各自的数字化管理系统。

近年来，国内博物馆数字化建设呈现以下特点：
- **政策支持力度加大**：国家出台了一系列政策文件，推动博物馆数字化建设
- **技术应用日益广泛**：云计算、大数据、AI等新技术在博物馆领域得到广泛应用
- **标准规范逐步完善**：制定了文物数字化采集、存储、管理等方面的标准规范
- **区域发展不平衡**：大型博物馆数字化水平较高，中小型博物馆仍处于起步阶段

**（3）存在的问题**

尽管国内外博物馆数字化建设取得了显著成果，但仍存在以下问题：
- **系统功能单一**：部分系统仅实现了基础的信息管理功能，缺乏3D展示、AI查询等高级功能
- **数据标准不统一**：不同博物馆采用的数据标准不一致，影响了数据的共享和交换
- **用户体验不佳**：部分系统界面设计陈旧，操作复杂，用户体验较差
- **安全性有待提高**：部分系统在数据安全、访问控制等方面存在隐患
- **维护成本较高**：系统的维护和升级需要专业技术人员，成本较高

### 1.2.2 文物管理系统的应用

文物管理系统在国内外博物馆中得到了广泛应用，主要应用场景包括：

**（1）藏品管理**
- 文物基本信息的录入、查询、修改和删除
- 文物分类管理和统计分析
- 文物图片、视频等多媒体资料的管理
- 文物状态跟踪（在库、借展、修复等）

**（2）借展管理**
- 借展申请的提交和审批
- 借展合同的管理

- 逾期提醒和催还功能

**（3）修复管理**
- 修复需求的评估和申请
- 修复专家的分配和管理
- 修复过程的记录和跟踪
- 修复前后对比和效果评估

**（4）展览管理**
- 展览策划和方案管理
- 展品的选择和布置
- 展览效果的评估和反馈

**（5）观众服务**
- 在线预约和购票
- 语音导览和虚拟参观
- 互动体验和教育活动
- 观众反馈和满意度调查

**（6）数据分析**
- 藏品统计和分析
- 观众流量分析
- 展览效果评估
- 运营数据分析

通过这些应用，文物管理系统有效提高了博物馆的管理效率，改善了观众的参观体验，促进了文化遗产的保护和传播。

## 2.1 功能需求

根据对博物馆文物管理业务的深入调研和分析，系统需要实现以下功能需求：

**（1）用户认证与权限管理**
- 用户注册、登录、登出功能
- 基于角色的访问控制（RBAC）
- 支持4种用户角色：系统管理员（ADMIN）、文物保管员（CURATOR）、借展审批员（APPROVER）、借展人（LOANER）
- 密码加密存储和密码重置功能
- 登录失败次数限制和账户锁定机制
- 操作日志记录和审计

**（2）文物信息管理**
- 文物基本信息的增删改查（CRUD）操作
- 文物编号自动生成
- 文物分类管理（支持树形结构）
- 文物状态管理（在库、借展中、修复中、封存）
- 文物图片管理（支持多图上传、主图设置）
- 文物3D模型管理（上传、查看、删除）

## ***\*3.3 系统核心算法设计\****
3.3.1 BCrypt密码加密算法
BCrypt是一种基于Blowfish加密算法的密码哈希函数，具有以下特点：

每次加密生成随机盐值
默认10轮迭代（2^10 = 1024次）
相同密码每次加密结果不同，防止彩虹表攻击
算法公式：

hash = BCrypt(password, salt, cost)
其中：

password：明文密码
salt：随机盐值（自动生成）
cost：迭代次数（默认10）
hash：加密后的密码哈希值
验证公式：

isMatch = BCrypt.checkpw(plainPassword, hashedPassword)
3.3.2 JWT认证机制
JWT（JSON Web Token）是一种无状态的认证机制，Token结构如下：

Token = Header.Payload.Signature
Header（头部）：

{
  "alg": "HS512",
  "typ": "JWT"
}
Payload（载荷）：

{
  "sub": "username",
  "iat": 1234567890,
  "exp": 1234654290
}
Signature（签名）：

Signature = HMACSHA512(
  base64UrlEncode(header) + "." + base64UrlEncode(payload),
  secret
)
3.3.3 AI智能查询相关度评分算法
相关度评分算法用于计算查询结果与查询关键词的匹配程度，总分100分。

评分公式：

总分 = 名称匹配分(40分) + 年代匹配分(24分) + 材质匹配分(24分) 
     + 分类匹配分(20分) + 描述匹配分(12分) + 来源匹配分(10分)
评分规则：

整句命中：名称+80分、编号+70分、描述+30分
关键词命中：名称+40分、编号+35分、年代+24分、材质+24分
同义词匹配：自动识别同义词并计分
相关度计算公式：

relevance = (score / maxScore) × 100
其中：

score：实际得分
maxScore：最高分（130分）
relevance：相关度（0-100）
3.3.4 图片智能提取算法
图片提取算法采用7层提取策略，按优先级依次尝试：

提取策略：

og:image meta标签
百度百科特定选择器（.summary-pic img）
class="summary-pic"图片
第一个有效img标签
data-src属性（懒加载）
srcset属性（响应式）
CSS背景图片
算法伪代码：

function extractImageUrl(doc):
    for strategy in strategies:
        url = strategy.extract(doc)
        if isValidImageUrl(url):
            return url
    return null
3.3.5 基于内容的协同过滤推荐算法
推荐算法用于推荐相关文物，基于文物的年代、材质、分类等属性进行匹配。

相似度计算公式：

similarity = w1 × era_match + w2 × material_match + w3 × category_match
其中：

w1 = 0.4（年代权重）
w2 = 0.3（材质权重）
w3 = 0.3（分类权重）
era_match：年代匹配度（0或1）
material_match：材质匹配度（0或1）
category_match：分类匹配度（0或1）
推荐算法：

recommendations = sort(relics, by=similarity, desc=True)
 

公式书写应在文中另起一行。公式之后应注明序号，该序号按章顺序编排。序号标注于同一行的最右边。较长的公式，另行居中横排。如公式必须转行时，只能在＋、－、×、÷处转行，上下式尽可能在“等号”处对齐。如第三章第二个公式应在其最右边标注“（3-2）”。

@RestController
@RequestMapping("/api/relics")
@Api(tags = "文物管理")
public class CulturalRelicController {
    
    @Autowired
    private CulturalRelicService culturalRelicService;
    
    /**
     * 分页查询文物列表
     */
    @GetMapping
    @ApiOperation("分页查询文物列表")
    @OperationLog(module = "文物管理", type = "查询", content = "查询文物列表")
    public Result<PageResult<CulturalRelic>> page(
            @RequestParam(defaultValue = "1") Integer pageNum,
            @RequestParam(defaultValue = "10") Integer pageSize,
            @RequestParam(required = false) String name,
            @RequestParam(required = false) Long categoryId,
            @RequestParam(required = false) String status) {
        
        PageResult<CulturalRelic> result = culturalRelicService.pageRelics(
            pageNum, pageSize, name, categoryId, status);
        return Result.success(result);
    }
    
    /**
     * 根据ID查询文物详情
     */
    @GetMapping("/{id}")
    @ApiOperation("查询文物详情")
    @OperationLog(module = "文物管理", type = "查询", content = "查询文物详情")
    public Result<CulturalRelic> getById(@PathVariable Long id) {
        CulturalRelic relic = culturalRelicService.getById(id);
        if (relic == null) {
            return Result.error("文物不存在");
        }
        return Result.success(relic);
    }
    
    /**
     * 添加文物
     */
    @PostMapping
    @ApiOperation("添加文物")
    @PreAuthorize("hasRole('CURATOR')")
    @OperationLog(module = "文物管理", type = "新增", content = "添加文物")
    public Result<Long> save(@RequestBody @Valid CulturalRelic relic) {
        // 自动生成文物编号
        String relicCode = culturalRelicService.generateRelicCode();
        relic.setRelicCode(relicCode);
        
        boolean success = culturalRelicService.save(relic);
        if (success) {
            return Result.success(relic.getId());
        }
        return Result.error("添加失败");
    }
    
    /**
     * 更新文物信息
     */
    @PutMapping("/{id}")
    @ApiOperation("更新文物信息")
    @PreAuthorize("hasRole('CURATOR')")
    @OperationLog(module = "文物管理", type = "修改", content = "更新文物信息")
    public Result<Void> update(@PathVariable Long id, 
                               @RequestBody @Valid CulturalRelic relic) {
        relic.setId(id);
        boolean success = culturalRelicService.updateById(relic);
        if (success) {
            return Result.success();
        }
        return Result.error("更新失败");
    }
    
    /**
     * 删除文物
     */
    @DeleteMapping("/{id}")
    @ApiOperation("删除文物")
    @PreAuthorize("hasRole('CURATOR')")
    @OperationLog(module = "文物管理", type = "删除", content = "删除文物")
    public Result<Void> delete(@PathVariable Long id) {
        // 检查文物状态
        CulturalRelic relic = culturalRelicService.getById(id);
        if (relic == null) {
            return Result.error("文物不存在");
        }
        if ("借展中".equals(relic.getStatus()) || "修复中".equals(relic.getStatus())) {
            return Result.error("文物正在借展或修复中，无法删除");
        }
        
        boolean success = culturalRelicService.removeById(id);
        if (success) {
            return Result.success();
        }
        return Result.error("删除失败");
    }
}

<template>
  <div class="relic-3d-view">
    <!-- 顶部信息栏 -->
    <div class="header">
      <div class="relic-info">
        <h2>{{ relic.relicName }}</h2>

    </div>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue'
import { useRoute, useRouter } from 'vue-router'
import { ElMessage } from 'element-plus'
import { ArrowLeft, Refresh, Aim, FullScreen, WarningFilled } from '@element-plus/icons-vue'
import * as THREE from 'three'
import { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader'
import { OBJLoader } from 'three/examples/jsm/loaders/OBJLoader'
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls'
import api from '@/api'

const route = useRoute()
const router = useRouter()

// 数据
const relic = ref({})
const loading = ref(true)
const error = ref('')
const containerRef = ref(null)
const autoRotate = ref(false)

// Three.js对象
let scene, camera, renderer, controls, model

// 获取文物信息
const fetchRelic = async () => {
  try {
    const response = await api.getRelic(route.params.id)
    relic.value = response.data
    
    if (!relic.value.model3dUrl) {
      error.value = '该文物没有3D模型'
      loading.value = false
      return
    }
    
    init3DScene()
    load3DModel()
  } catch (err) {
    error.value = '加载文物信息失败'
    loading.value = false
  }
}

// 初始化3D场景
const init3DScene = () => {
  const container = containerRef.value
  const width = container.clientWidth
  const height = container.clientHeight
  
  // 创建场景
  scene = new THREE.Scene()
  scene.background = new THREE.Color(0xf0f0f0)
  
  // 创建相机
  camera = new THREE.PerspectiveCamera(75, width / height, 0.1, 1000)
  camera.position.set(0, 0, 5)
  
  // 创建渲染器
  renderer = new THREE.WebGLRenderer({ antialias: true })
  renderer.setSize(width, height)
  renderer.setPixelRatio(window.devicePixelRatio)
  container.appendChild(renderer.domElement)
  
  // 添加光照
  const ambientLight = new THREE.AmbientLight(0xffffff, 0.6)
  scene.add(ambientLight)
  
  const directionalLight = new THREE.DirectionalLight(0xffffff, 0.8)
  directionalLight.position.set(10, 10, 10)
  scene.add(directionalLight)
  
  // 添加轨道控制器
  controls = new OrbitControls(camera, renderer.domElement)
  controls.enableDamping = true
  controls.dampingFactor = 0.05
  controls.minDistance = 1
  controls.maxDistance = 50
  
  // 监听窗口大小变化
  window.addEventListener('resize', onWindowResize)
  
  // 启动渲染循环
  animate()
}

// 加载3D模型
const load3DModel = () => {
  const modelUrl = relic.value.model3dUrl
  const fileExtension = modelUrl.split('.').pop().toLowerCase()
  
  let loader
  if (fileExtension === 'gltf' || fileExtension === 'glb') {
    loader = new GLTFLoader()
  } else if (fileExtension === 'obj') {
    loader = new OBJLoader()
  } else {
    error.value = '不支持的模型格式'
    loading.value = false
    return
  }
  
  loader.load(
    modelUrl,
    (object) => {
      // 加载成功
      model = fileExtension === 'obj' ? object : object.scene
      
      // 计算模型边界框
      const box = new THREE.Box3().setFromObject(model)
      const center = box.getCenter(new THREE.Vector3())
      const size = box.getSize(new THREE.Vector3())
      
      // 模型居中
      model.position.sub(center)
      
      // 自动缩放
      const maxDim = Math.max(size.x, size.y, size.z)
      const scale = 3 / maxDim
      model.scale.setScalar(scale)
      
      // 添加到场景
      scene.add(model)
      
      loading.value = false
    },
    (xhr) => {
      // 加载进度
      const percent = (xhr.loaded / xhr.total) * 100
      console.log(`加载进度: ${percent.toFixed(2)}%`)
    },
    (err) => {
      // 加载失败
      console.error('加载模型失败:', err)
      error.value = '加载3D模型失败'
      loading.value = false
    }
  )
}

// 渲染循环
const animate = () => {
  requestAnimationFrame(animate)
  
  // 自动旋转
  if (autoRotate.value && model) {
    model.rotation.y += 0.01
  }
  
  controls.update()
  renderer.render(scene, camera)
}

// 窗口大小变化
const onWindowResize = () => {
  const container = containerRef.value
  const width = container.clientWidth
  const height = container.clientHeight
  
  camera.aspect = width / height
  camera.updateProjectionMatrix()
  renderer.setSize(width, height)
}

// 切换自动旋转
const toggleAutoRotate = () => {
  autoRotate.value = !autoRotate.value
}

// 重置相机
const resetCamera = () => {
  camera.position.set(0, 0, 5)
  controls.reset()
}

// 全屏
const toggleFullscreen = () => {
  const container = containerRef.value
  if (!document.fullscreenElement) {
    container.requestFullscreen()
  } else {
    document.exitFullscreen()
  }
}

// 返回
const goBack = () => {
  const query = route.query
  router.push({
    path: '/relics',
    query: {
      page: query.page,
      pageSize: query.pageSize,
      name: query.name,
      categoryId: query.categoryId,
      status: query.status
    }
  })
}

// 清理资源
const cleanup = () => {
  window.removeEventListener('resize', onWindowResize)
  
  if (renderer) {
    renderer.dispose()
  }
  
  if (scene) {
    scene.traverse((object) => {
      if (object.geometry) {
        object.geometry.dispose()
      }
      if (object.material) {
        if (Array.isArray(object.material)) {
          object.material.forEach(material => material.dispose())
        } else {
          object.material.dispose()
        }
      }
    })
  }
}

// 生命周期
onMounted(() => {
  fetchRelic()
})

onUnmounted(() => {
  cleanup()
})
</script>

@Service
@Slf4j
public class AiQueryServiceImpl implements AiQueryService {
    
    @Autowired
    private CulturalRelicMapper culturalRelicMapper;
    
    @Autowired
    private AiChatMessageMapper aiChatMessageMapper;
    
    @Autowired
    private AiQueryResultMapper aiQueryResultMapper;
    
    /**
     * AI智能查询
     */
    @Override
    public AiRelicQueryResponse query(AiRelicQueryRequest request) {
        String query = request.getQuery();
        Long sessionId = request.getSessionId();
        
        log.info("AI查询开始，关键词：{}", query);
        
        // 1. 关键词预处理
        List<String> keywords = preprocessKeywords(query);
        log.info("关键词预处理结果：{}", keywords);
        
        // 2. 第一层：馆藏检索
        List<AiRelicItemVO> museumResults = searchMuseumRelics(keywords);
        log.info("馆藏检索结果数量：{}", museumResults.size());
        
        // 3. 第二层：全网检索（如果馆藏结果不足3个）
        List<AiRelicItemVO> webResults = new ArrayList<>();
        if (museumResults.size() < 3) {
            webResults = searchWebRelics(keywords);
            log.info("全网检索结果数量：{}", webResults.size());
        }
        
        // 4. 合并结果
        List<AiRelicItemVO> allResults = new ArrayList<>();
        allResults.addAll(museumResults);
        allResults.addAll(webResults);
        
        // 5. 按相关度排序
        allResults.sort((a, b) -> b.getRelevance().compareTo(a.getRelevance()));
        
        // 6. 保存查询结果
        saveQueryResults(sessionId, query, allResults);
        
        // 7. 构建响应
        AiRelicQueryResponse response = new AiRelicQueryResponse();
        response.setQuery(query);
        response.setResults(allResults);
        response.setTotalCount(allResults.size());
        response.setMuseumCount(museumResults.size());
        response.setWebCount(webResults.size());
        
        log.info("AI查询完成，总结果数：{}", allResults.size());
        
        return response;
    }
    
    /**
     * 关键词预处理
     */
    private List<String> preprocessKeywords(String query) {
        List<String> keywords = new ArrayList<>();
        
        // 简单分词
        String[] words = query.split("[\\s,，。.]+");
        
        for (String word : words) {
            if (word.length() > 0) {
                keywords.add(word);
                // 添加同义词
                keywords.addAll(getSynonyms(word));
            }
        }
        
        return keywords.stream().distinct().collect(Collectors.toList());
    }
    
    /**
     * 获取同义词
     */
    private List<String> getSynonyms(String word) {
        Map<String, List<String>> synonymMap = new HashMap<>();
        synonymMap.put("青铜器", Arrays.asList("铜器", "青铜"));
        synonymMap.put("陶瓷", Arrays.asList("瓷器", "陶器", "瓷"));
        synonymMap.put("玉器", Arrays.asList("玉", "玉石"));
        synonymMap.put("书画", Arrays.asList("字画", "画作", "书法"));
        
        return synonymMap.getOrDefault(word, Collections.emptyList());
    }
    
    /**
     * 馆藏检索
     */
    private List<AiRelicItemVO> searchMuseumRelics(List<String> keywords) {
        List<CulturalRelic> relics = culturalRelicMapper.searchByKeywords(keywords);
        
        return relics.stream()
            .map(relic -> {
                AiRelicItemVO item = new AiRelicItemVO();
                item.setId(relic.getId());
                item.setName(relic.getRelicName());
                item.setCode(relic.getRelicCode());
                item.setEra(relic.getEra());
                item.setMaterial(relic.getMaterial());
                item.setCategoryName(relic.getCategoryName());
                item.setDescription(relic.getDescription());
                item.setImageUrl(relic.getImagePath());
                item.setSourceType("馆藏");
                item.setSourceName("本馆");
                
                // 计算相关度
                int relevance = calculateRelevance(relic, keywords);
                item.setRelevance(relevance);
                
                return item;
            })
            .filter(item -> item.getRelevance() > 0)
            .collect(Collectors.toList());
    }
    
    /**
     * 计算相关度评分
     */
    private int calculateRelevance(CulturalRelic relic, List<String> keywords) {
        int score = 0;
        
        for (String keyword : keywords) {
            // 名称匹配（40分）
            if (relic.getRelicName() != null && 
                relic.getRelicName().contains(keyword)) {
                score += 40;
            }
            
            // 编号匹配（35分）
            if (relic.getRelicCode() != null && 
                relic.getRelicCode().contains(keyword)) {
                score += 35;
            }
            
            // 年代匹配（24分）
            if (relic.getEra() != null && 
                relic.getEra().contains(keyword)) {
                score += 24;
            }
            
            // 材质匹配（24分）
            if (relic.getMaterial() != null && 
                relic.getMaterial().contains(keyword)) {
                score += 24;
            }
            
            // 分类匹配（20分）
            if (relic.getCategoryName() != null && 
                relic.getCategoryName().contains(keyword)) {
                score += 20;
            }
            
            // 描述匹配（12分）
            if (relic.getDescription() != null && 
                relic.getDescription().contains(keyword)) {
                score += 12;
            }
            
            // 来源匹配（10分）
            if (relic.getOrigin() != null && 
                relic.getOrigin().contains(keyword)) {
                score += 10;
            }
        }
        
        // 归一化到0-100
        return Math.min(100, (int)((score / 130.0) * 100));
    }
    
    /**
     * 全网检索
     */
    private List<AiRelicItemVO> searchWebRelics(List<String> keywords) {
        List<AiRelicItemVO> results = new ArrayList<>();
        
        try {
            // 构建搜索URL
            String searchQuery = String.join("+", keywords);
            String searchUrl = "https://baike.baidu.com/search?word=" + 
                URLEncoder.encode(searchQuery, "UTF-8");
            
            // 爬取搜索结果页面
            Document doc = Jsoup.connect(searchUrl)
                .userAgent("Mozilla/5.0")
                .timeout(5000)
                .get();
            
            // 提取搜索结果
            Elements items = doc.select(".result-item");
            
            for (Element item : items) {
                try {
                    AiRelicItemVO webItem = extractWebItem(item, keywords);
                    if (webItem != null) {
                        results.add(webItem);
                    }
                } catch (Exception e) {
                    log.error("提取网页结果失败", e);
                }
                
                if (results.size() >= 5) break;
            }
        } catch (Exception e) {
            log.error("全网检索失败", e);
        }
        
        return results;
    }
    
    /**
     * 提取网页结果
     */
    private AiRelicItemVO extractWebItem(Element item, List<String> keywords) {
        AiRelicItemVO webItem = new AiRelicItemVO();
        
        // 提取标题
        Element titleElem = item.selectFirst(".title");
        if (titleElem != null) {
            webItem.setName(titleElem.text());
        }
        
        // 提取描述
        Element descElem = item.selectFirst(".description");
        if (descElem != null) {
            webItem.setDescription(descElem.text());
        }
        
        // 提取图片
        String imageUrl = extractImageUrl(item);
        webItem.setImageUrl(imageUrl);
        
        // 设置来源
        webItem.setSourceType("外部");
        webItem.setSourceName("百度百科");
        
        // 计算相关度
        int relevance = calculateWebRelevance(webItem, keywords);
        webItem.setRelevance(relevance);
        
        return webItem;
    }
    
    /**
     * 提取图片URL（7层策略）
     */
    private String extractImageUrl(Element element) {
        // 1. og:image meta标签
        Element ogImage = element.selectFirst("meta[property=og:image]");
        if (ogImage != null) {
            return ogImage.attr("content");
        }
        
        // 2. 百度百科特定选择器
        Element baikeImg = element.selectFirst(".summary-pic img");
        if (baikeImg != null) {
            return baikeImg.attr("src");
        }
        
        // 3. class="summary-pic"图片
        Element summaryPic = element.selectFirst("img.summary-pic");
        if (summaryPic != null) {
            return summaryPic.attr("src");
        }
        
        // 4. 第一个有效img标签
        Elements imgs = element.select("img[src]");
        for (Element img : imgs) {
            String src = img.attr("src");
            if (isValidImageUrl(src)) {
                return src;
            }
        }
        
        // 5. data-src属性（懒加载）
        Element lazyImg = element.selectFirst("img[data-src]");
        if (lazyImg != null) {
            return lazyImg.attr("data-src");
        }
        
        // 6. srcset属性（响应式）
        Element srcsetImg = element.selectFirst("img[srcset]");
        if (srcsetImg != null) {
            String srcset = srcsetImg.attr("srcset");
            return parseSrcset(srcset);
        }
        
        // 7. CSS背景图片
        return extractBackgroundImage(element);
    }
    
    /**
     * 验证图片URL
     */
    private boolean isValidImageUrl(String url) {
        if (url == null || url.isEmpty()) {
            return false;
        }
        
        String lowerUrl = url.toLowerCase();
        return lowerUrl.endsWith(".jpg") || 
               lowerUrl.endsWith(".jpeg") || 
               lowerUrl.endsWith(".png") || 
               lowerUrl.endsWith(".webp");
    }
    
    /**
     * 保存查询结果
     */
    private void saveQueryResults(Long sessionId, String query, 
                                  List<AiRelicItemVO> results) {
        // 保存消息
        AiChatMessage message = new AiChatMessage();
        message.setSessionId(sessionId);
        message.setRole("assistant");
        message.setContent("为您找到" + results.size() + "个相关结果");
        aiChatMessageMapper.insert(message);
        
        // 保存查询结果
        for (AiRelicItemVO item : results) {
            AiQueryResult result = new AiQueryResult();
            result.setMessageId(message.getId());
            result.setRelicId(item.getId());
            result.setRelevance(item.getRelevance());
            result.setSourceType(item.getSourceType());
            result.setSourceName(item.getSourceName());
            aiQueryResultMapper.insert(result);
        }
    }
}

<script setup>
import { ref, reactive, onMounted, nextTick } from 'vue'
import { ElMessage, ElMessageBox } from 'element-plus'
import { Plus, Delete, User, Robot, Loading, Search } from '@element-plus/icons-vue'
import { useRouter } from 'vue-router'
import api from '@/api'
import dayjs from 'dayjs'

const router = useRouter()

// 数据
const sessions = ref([])
const currentSessionId = ref(null)
const messages = ref([])
const inputText = ref('')
const loading = ref(false)
const messagesRef = ref(null)

// 获取会话列表
const fetchSessions = async () => {
  try {
    const response = await api.getAiSessions()
    sessions.value = response.data
    
    if (sessions.value.length > 0 && !currentSessionId.value) {
      currentSessionId.value = sessions.value[0].id
      fetchMessages(currentSessionId.value)
    }
  } catch (error) {
    ElMessage.error('获取会话列表失败')
  }
}

// 获取消息列表
const fetchMessages = async (sessionId) => {
  try {
    const response = await api.getAiMessages(sessionId)
    messages.value = response.data
    scrollToBottom()
  } catch (error) {
    ElMessage.error('获取消息列表失败')
  }
}

// 创建会话
const createSession = async () => {
  try {
    const response = await api.createAiSession({
      title: '新会话'
    })
    sessions.value.unshift(response.data)
    currentSessionId.value = response.data.id
    messages.value = []
  } catch (error) {
    ElMessage.error('创建会话失败')
  }
}

// 选择会话
const selectSession = (sessionId) => {
  currentSessionId.value = sessionId
  fetchMessages(sessionId)
}

// 删除会话
const deleteSession = async (sessionId) => {
  try {
    await ElMessageBox.confirm('确定要删除该会话吗？', '提示', {
      confirmButtonText: '确定',
      cancelButtonText: '取消',
      type: 'warning'
    })
    
    await api.deleteAiSession(sessionId)
    sessions.value = sessions.value.filter(s => s.id !== sessionId)
    
    if (currentSessionId.value === sessionId) {
      currentSessionId.value = sessions.value[0]?.id || null
      if (currentSessionId.value) {
        fetchMessages(currentSessionId.value)
      } else {
        messages.value = []
      }
    }
    
    ElMessage.success('删除成功')
  } catch (error) {
    if (error !== 'cancel') {
      ElMessage.error('删除失败')
    }
  }
}

// 发送查询
const handleSend = async () => {
  if (!inputText.value.trim()) {
    ElMessage.warning('请输入查询内容')
    return
  }
  
  if (!currentSessionId.value) {
    await createSession()
  }
  
  const query = inputText.value.trim()
  inputText.value = ''
  
  // 添加用户消息
  messages.value.push({
    role: 'user',
    content: query
  })
  scrollToBottom()
  
  // 发送查询请求
  loading.value = true
  try {
    const response = await api.queryAiRelics({
      sessionId: currentSessionId.value,
      query: query
    })
    
    // 添加AI响应
    messages.value.push({
      role: 'assistant',
      content: `为您找到${response.data.totalCount}个相关结果（馆藏${response.data.museumCount}个，外部${response.data.webCount}个）`,
      results: response.data.results
    })
    
    scrollToBottom()
  } catch (error) {
    ElMessage.error('查询失败')
  } finally {
    loading.value = false
  }
}

// 查看文物详情
const viewRelic = (result) => {
  if (result.id) {
    router.push(`/relics/${result.id}`)
  } else {
    ElMessage.info('外部文物暂不支持查看详情')
  }
}

// 滚动到底部
const scrollToBottom = () => {
  nextTick(() => {
    if (messagesRef.value) {
      messagesRef.value.scrollTop = messagesRef.value.scrollHeight
    }
  })
}

// 格式化时间
const formatTime = (time) => {
  return dayjs(time).format('MM-DD HH:mm')
}

// 初始化
onMounted(() => {
  fetchSessions()
})
</script>

@Service
@Slf4j
public class StatisticsServiceImpl implements StatisticsService {
    
    @Autowired
    private CulturalRelicMapper culturalRelicMapper;
    
    @Autowired
    private RelicArchiveMapper relicArchiveMapper;
    
    @Override
    public Long countTotalRelics() {
        return culturalRelicMapper.selectCount(null);
    }
    
    @Override
    public Long countRelicsByStatus(String status) {
        QueryWrapper<CulturalRelic> wrapper = new QueryWrapper<>();
        wrapper.eq("status", status);
        return culturalRelicMapper.selectCount(wrapper);
    }
    
    @Override
    public Long countTotalArchives() {
        return relicArchiveMapper.selectCount(null);
    }
    
    @Override
    public List<Map<String, Object>> getCategoryStats() {
        return culturalRelicMapper.selectCategoryStats();
    }
    
    @Override
    public Map<String, Long> getStatusDistribution() {
        List<CulturalRelic> relics = culturalRelicMapper.selectList(null);
        
        return relics.stream()
            .collect(Collectors.groupingBy(
                CulturalRelic::getStatus,
                Collectors.counting()
            ));
    }
    
    @Override
    public List<Map<String, Object>> getEraDistribution() {
        return culturalRelicMapper.selectEraDistribution();
    }
    
    @Override
    public List<Map<String, Object>> getMaterialDistribution() {
        return culturalRelicMapper.selectMaterialDistribution();
    }

@RestController
@RequestMapping("/api/users")
@Api(tags = "用户管理")
public class SysUserController {
    
    @Autowired
    private SysUserService sysUserService;
    
    /**
     * 分页查询用户列表
     */
    @GetMapping
    @ApiOperation("分页查询用户列表")
    @PreAuthorize("hasRole('ADMIN')")
    @OperationLog(module = "用户管理", type = "查询", content = "查询用户列表")
    public Result<PageResult<SysUser>> page(
            @RequestParam(defaultValue = "1") Integer pageNum,
            @RequestParam(defaultValue = "10") Integer pageSize,
            @RequestParam(required = false) String username,
            @RequestParam(required = false) String realName,
            @RequestParam(required = false) String role) {
        
        PageResult<SysUser> result = sysUserService.pageUsers(
            pageNum, pageSize, username, realName, role);
        return Result.success(result);
    }
    
    /**
     * 添加用户
     */
    @PostMapping
    @ApiOperation("添加用户")
    @PreAuthorize("hasRole('ADMIN')")
    @OperationLog(module = "用户管理", type = "新增", content = "添加用户")
    public Result<Long> save(@RequestBody @Valid SysUser user) {
        // 检查用户名是否已存在
        SysUser existUser = sysUserService.getByUsername(user.getUsername());
        if (existUser != null) {
            return Result.error("用户名已存在");
        }
        
        // 加密密码
        String encryptedPassword = BCrypt.hashpw(user.getPassword(), BCrypt.gensalt());
        user.setPassword(encryptedPassword);
        user.setStatus(1); // 默认启用
        
        boolean success = sysUserService.save(user);
        if (success) {
            return Result.success(user.getId());
        }
        return Result.error("添加失败");
    }
    
    /**
     * 更新用户信息
     */
    @PutMapping("/{id}")
    @ApiOperation("更新用户信息")
    @PreAuthorize("hasRole('ADMIN')")
    @OperationLog(module = "用户管理", type = "修改", content = "更新用户信息")
    public Result<Void> update(@PathVariable Long id, 
                               @RequestBody @Valid SysUser user) {
        user.setId(id);
        // 不更新密码
        user.setPassword(null);
        
        boolean success = sysUserService.updateById(user);
        if (success) {
            return Result.success();
        }
        return Result.error("更新失败");
    }
    
    /**
     * 删除用户
     */
    @DeleteMapping("/{id}")
    @ApiOperation("删除用户")
    @PreAuthorize("hasRole('ADMIN')")
    @OperationLog(module = "用户管理", type = "删除", content = "删除用户")
    public Result<Void> delete(@PathVariable Long id) {
        // 不能删除自己
        if (id.equals(SecurityUtils.getUserId())) {
            return Result.error("不能删除当前登录用户");
        }
        
        boolean success = sysUserService.removeById(id);
        if (success) {
            return Result.success();
        }
        return Result.error("删除失败");
    }
    
    /**
     * 重置密码
     */
    @PutMapping("/{id}/reset-password")
    @ApiOperation("重置密码")
    @PreAuthorize("hasRole('ADMIN')")
    @OperationLog(module = "用户管理", type = "修改", content = "重置密码")
    public Result<Void> resetPassword(@PathVariable Long id) {
        // 重置为默认密码：123456
        String defaultPassword = "123456";
        String encryptedPassword = BCrypt.hashpw(defaultPassword, BCrypt.gensalt());
        
        SysUser user = new SysUser();
        user.setId(id);
        user.setPassword(encryptedPassword);
        user.setLoginFailedCount(0); // 重置失败次数
        user.setAccountLocked(0); // 解除锁定
        
        boolean success = sysUserService.updateById(user);
        if (success) {
            return Result.success();
        }
        return Result.error("重置失败");
    }
}

    @Override
    public PageResult<SysUser> pageUsers(Integer pageNum, Integer pageSize, 
                                        String username, String realName, String role) {
        Page<SysUser> page = new Page<>(pageNum, pageSize);
        
        QueryWrapper<SysUser> wrapper = new QueryWrapper<>();
        if (StringUtils.isNotBlank(username)) {
            wrapper.like("username", username);
        }
        if (StringUtils.isNotBlank(realName)) {
            wrapper.like("real_name", realName);
        }
        if (StringUtils.isNotBlank(role)) {
            wrapper.eq("role", role);
        }
        wrapper.orderByDesc("create_time");
        
        IPage<SysUser> result = this.page(page, wrapper);
        
        // 清除密码字段
        result.getRecords().forEach(user -> user.setPassword(null));
        
        return new PageResult<>(
            result.getRecords(),
            result.getTotal(),
            result.getPages(),
            result.getCurrent()
        );
    }
    
    @Override
    public SysUser getByUsername(String username) {
        QueryWrapper<SysUser> wrapper = new QueryWrapper<>();
        wrapper.eq("username", username);
        return this.getOne(wrapper);
    }

@RestController
@RequestMapping("/api/repairs")
@Api(tags = "修复管理")
public class RepairRecordController {
    
    @Autowired
    private RepairRecordService repairRecordService;
    
    @Autowired
    private CulturalRelicService culturalRelicService;
    
    @Autowired
    private NotificationService notificationService;
    
    /**
     * 分页查询修复记录
     */
    @GetMapping
    @ApiOperation("分页查询修复记录")
    @OperationLog(module = "修复管理", type = "查询", content = "查询修复记录列表")
    public Result<PageResult<RepairRecord>> page(
            @RequestParam(defaultValue = "1") Integer pageNum,
            @RequestParam(defaultValue = "10") Integer pageSize,
            @RequestParam(required = false) String relicName,
            @RequestParam(required = false) String status) {
        
        PageResult<RepairRecord> result = repairRecordService.pageRecords(
            pageNum, pageSize, relicName, status);
        return Result.success(result);
    }
    
    /**
     * 申请修复
     */
    @PostMapping
    @ApiOperation("申请修复")
    @PreAuthorize("hasRole('CURATOR')")
    @OperationLog(module = "修复管理", type = "新增", content = "申请修复")
    public Result<Long> apply(@RequestBody @Valid RepairApplyRequest request) {
        // 检查文物状态
        CulturalRelic relic = culturalRelicService.getById(request.getRelicId());
        if (relic == null) {
            return Result.error("文物不存在");
        }
        if (!"在库".equals(relic.getStatus())) {
            return Result.error("该文物当前不可申请修复");
        }
        
        // 创建修复记录
        RepairRecord repairRecord = new RepairRecord();
        repairRecord.setRelicId(request.getRelicId());
        repairRecord.setExpertId(request.getExpertId());
        repairRecord.setRepairType(request.getRepairType());
        repairRecord.setDescription(request.getDescription());
        repairRecord.setPriority(request.getPriority());
        repairRecord.setStatus("待审批");
        
        boolean success = repairRecordService.save(repairRecord);
        if (success) {
            // 发送通知给审批员
            notificationService.notifyApprovers(
                "修复申请",
                "收到新的修复申请：" + relic.getRelicName(),
                repairRecord.getId()
            );
            
            return Result.success(repairRecord.getId());
        }
        return Result.error("申请失败");
    }
    
    /**
     * 审批修复
     */
    @PutMapping("/{id}/approve")
    @ApiOperation("审批修复")
    @PreAuthorize("hasRole('APPROVER')")
    @OperationLog(module = "修复管理", type = "修改", content = "审批修复")
    public Result<Void> approve(@PathVariable Long id, 
                               @RequestBody @Valid RepairApproveRequest request) {
        RepairRecord repairRecord = repairRecordService.getById(id);
        if (repairRecord == null) {
            return Result.error("修复记录不存在");
        }
        
        if (!"待审批".equals(repairRecord.getStatus())) {
            return Result.error("该修复记录已处理");
        }
        
        // 更新修复记录
        repairRecord.setStatus(request.isApproved() ? "待修复" : "已拒绝");
        repairRecord.setApproverName(SecurityUtils.getUsername());
        repairRecord.setApproveTime(new Date());
        repairRecord.setApproveRemark(request.getRemark());
        
        boolean success = repairRecordService.updateById(repairRecord);
        
        if (success && request.isApproved()) {
            // 更新文物状态
            CulturalRelic relic = culturalRelicService.getById(repairRecord.getRelicId());
            relic.setStatus("修复中");
            culturalRelicService.updateById(relic);
            
            // 发送通知给专家
            notificationService.notifyExpert(
                repairRecord.getExpertId(),
                "修复任务",
                "您有新的修复任务：" + relic.getRelicName(),
                repairRecord.getId()
            );
        }
        
        return Result.success();
    }
    
    /**
     * 更新修复进度
     */
    @PutMapping("/{id}/progress")
    @ApiOperation("更新修复进度")
    @PreAuthorize("hasRole('CURATOR')")
    @OperationLog(module = "修复管理", type = "修改", content = "更新修复进度")
    public Result<Void> updateProgress(@PathVariable Long id, 
                                      @RequestBody @Valid RepairProgressRequest request) {
        RepairRecord repairRecord = repairRecordService.getById(id);
        if (repairRecord == null) {
            return Result.error("修复记录不存在");
        }
        
        // 更新修复记录
        repairRecord.setStatus(request.getStatus());
        repairRecord.setProgress(request.getProgress());
        repairRecord.setProgressDescription(request.getDescription());
        
        if ("已完成".equals(request.getStatus())) {
            repairRecord.setEndDate(new Date());
            repairRecord.setResult(request.getResult());
            
            // 更新文物状态
            CulturalRelic relic = culturalRelicService.getById(repairRecord.getRelicId());
            relic.setStatus("在库");
            culturalRelicService.updateById(relic);
        }
        
        boolean success = repairRecordService.updateById(repairRecord);
        if (success) {
            return Result.success();
        }
        return Result.error("更新失败");
    }
}

    @Autowired
    private RepairRecordMapper repairRecordMapper;
    
    @Override
    public String generateExpertCode() {
        // 查询最大编号
        QueryWrapper<RepairExpert> wrapper = new QueryWrapper<>();
        wrapper.orderByDesc("expert_code").last("LIMIT 1");
        RepairExpert lastExpert = this.getOne(wrapper);
        
        if (lastExpert == null || lastExpert.getExpertCode() == null) {
            return "EXP00001";
        }
        
        // 提取数字部分并加1
        String lastCode = lastExpert.getExpertCode();
        int number = Integer.parseInt(lastCode.substring(3)) + 1;
        return String.format("EXP%05d", number);
    }
    
    @Override
    public boolean hasActiveRepairs(Long expertId) {
        QueryWrapper<RepairRecord> wrapper = new QueryWrapper<>();
        wrapper.eq("expert_id", expertId);
        wrapper.in("status", Arrays.asList("待修复", "修复中"));
        
        return repairRecordMapper.selectCount(wrapper) > 0;
    }
    
    @Override
    public PageResult<RepairExpert> pageExperts(Integer pageNum, Integer pageSize, 
                                                String name, String specialty) {
        Page<RepairExpert> page = new Page<>(pageNum, pageSize);
        
        QueryWrapper<RepairExpert> wrapper = new QueryWrapper<>();
        if (StringUtils.isNotBlank(name)) {
            wrapper.like("expert_name", name);
        }
        if (StringUtils.isNotBlank(specialty)) {
            wrapper.like("specialty", specialty);
        }
        wrapper.orderByDesc("create_time");
        
        IPage<RepairExpert> result = this.page(page, wrapper);
        
        return new PageResult<>(
            result.getRecords(),
            result.getTotal(),
            result.getPages(),

            log.info("材料出库：{}, 数量：{}, 当前库存：{}", 
                    material.getMaterialName(), quantity, material.getStockQuantity());
        }
        
        return success;
    }

@Aspect
@Component
@Slf4j
public class OperationLogAspect {
    
    @Autowired
    private SysOperationLogService sysOperationLogService;
    
    @Autowired
    private HttpServletRequest request;
    
    /**
     * 环绕通知，记录操作日志
     */
    @Around("@annotation(operationLog)")
    public Object around(ProceedingJoinPoint joinPoint, OperationLog operationLog) throws Throwable {
        // 记录开始时间
        long startTime = System.currentTimeMillis();
        
        // 构建日志对象
        SysOperationLog log = new SysOperationLog();
        log.setModule(operationLog.module());
        log.setType(operationLog.type());
        log.setContent(operationLog.content());
        log.setOperator(SecurityUtils.getUsername());
        log.setOperatorId(SecurityUtils.getUserId());
        log.setIpAddress(getClientIp());
        log.setOperationTime(new Date());
        
        // 记录请求参数
        Object[] args = joinPoint.getArgs();
        if (args != null && args.length > 0) {
            try {
                String params = JSON.toJSONString(args);
                log.setRequestParams(params);
            } catch (Exception e) {
                log.setRequestParams("参数序列化失败");
            }
        }
        
        try {
            // 执行目标方法
            Object result = joinPoint.proceed();
            
            // 记录成功
            log.setStatus("成功");
            log.setExecutionTime(System.currentTimeMillis() - startTime);
            
            // 记录响应结果
            try {
                String response = JSON.toJSONString(result);
                if (response.length() > 1000) {
                    response = response.substring(0, 1000) + "...";
                }
                log.setResponseData(response);
            } catch (Exception e) {
                log.setResponseData("响应序列化失败");
            }
            
            // 异步保存日志
            saveLogAsync(log);
            
            return result;
        } catch (Throwable e) {
            // 记录失败
            log.setStatus("失败");
            log.setExecutionTime(System.currentTimeMillis() - startTime);
            log.setErrorMessage(e.getMessage());
            
            // 记录异常堆栈（前500个字符）
            String stackTrace = getStackTrace(e);
            if (stackTrace.length() > 500) {
                stackTrace = stackTrace.substring(0, 500) + "...";
            }
            log.setStackTrace(stackTrace);
            
            // 异步保存日志
            saveLogAsync(log);
            
            throw e;
        }
    }
    
    /**
     * 异步保存日志
     */
    @Async
    public void saveLogAsync(SysOperationLog log) {
        try {
            sysOperationLogService.save(log);
        } catch (Exception e) {
            log.error("保存操作日志失败", e);
        }
    }
    
    /**
     * 获取客户端IP地址
     */
    private String getClientIp() {
        String ip = request.getHeader("X-Forwarded-For");
        if (ip == null || ip.length() == 0 || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("X-Real-IP");
        }
        if (ip == null || ip.length() == 0 || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getRemoteAddr();
        }
        return ip;
    }
    
    /**
     * 获取异常堆栈信息
     */
    private String getStackTrace(Throwable throwable) {
        StringWriter sw = new StringWriter();
        PrintWriter pw = new PrintWriter(sw);
        throwable.printStackTrace(pw);
        return sw.toString();
    }
}

        try {
            // 生成备份文件名
            String timestamp = new SimpleDateFormat("yyyyMMdd_HHmmss").format(new Date());

            );

            if (exitCode == 0) {
                // 获取文件大小

                }

                record.setFileName(fileName);
                record.setFilePath(backupPath);
                record.setFileSize(fileSize);
                record.setBackupType("手动");
                record.setBackupTime(new Date());
                record.setStatus("成功");
                record.setEncrypted(encrypted != null && encrypted);

            }

        try {
            BackupRecord record = backupService.getById(id);
            if (record == null) {
                response.setStatus(HttpServletResponse.SC_NOT_FOUND);
                return;
            }
            
            File file = new File(record.getFilePath());
            if (!file.exists()) {
                response.setStatus(HttpServletResponse.SC_NOT_FOUND);
                return;
            }
            
            response.setContentType("application/octet-stream");
            response.setHeader("Content-Disposition", 
                "attachment; filename=" + URLEncoder.encode(record.getFileName(), "UTF-8"));
            
            try (FileInputStream fis = new FileInputStream(file);
                 OutputStream os = response.getOutputStream()) {
                byte[] buffer = new byte[1024];
                int len;
                while ((len = fis.read(buffer)) != -1) {
                    os.write(buffer, 0, len);
                }
                os.flush();
            }
        } catch (Exception e) {
            log.error("下载备份文件失败", e);
        }
    }

        QueryWrapper<AiChatMessage> wrapper = new QueryWrapper<>();
        wrapper.eq("session_id", sessionId);
        wrapper.orderByAsc("create_time");
        
        List<AiChatMessage> messages = this.list(wrapper);
        
        // 转换为VO并加载查询结果
        return messages.stream().map(message -> {
            AiChatMessageVO vo = new AiChatMessageVO();
            vo.setId(message.getId());
            vo.setSessionId(message.getSessionId());
            vo.setRole(message.getRole());
            vo.setContent(message.getContent());
            vo.setCreateTime(message.getCreateTime());
            
            // 如果是assistant消息，加载查询结果
            if ("assistant".equals(message.getRole())) {
                QueryWrapper<AiQueryResult> resultWrapper = new QueryWrapper<>();
                resultWrapper.eq("message_id", message.getId());
                List<AiQueryResult> results = aiQueryResultMapper.selectList(resultWrapper);
                
                // 转换为VO
                List<AiRelicItemVO> resultVOs = results.stream().map(result -> {
                    AiRelicItemVO itemVO = new AiRelicItemVO();
                    // 设置查询结果属性
                    itemVO.setId(result.getRelicId());
                    itemVO.setRelevance(result.getRelevance());
                    itemVO.setSourceType(result.getSourceType());
                    itemVO.setSourceName(result.getSourceName());
                    // 其他属性从文物表查询
                    return itemVO;
                }).collect(Collectors.toList());
                
                vo.setResults(resultVOs);
            }
            
            return vo;
        }).collect(Collectors.toList());
    }

5.2.1 系统简介
博物馆文物数字化管理系统是一个基于B/S架构的Web应用系统,采用Spring Boot + Vue 3技术栈开发,旨在为博物馆提供全面的文物数字化管理解决方案。

系统主要功能包括:

文物信息管理: 文物的增删改查、分类管理、批量操作、Excel导入导出
3D文物展示: 基于Three.js的3D模型展示,支持交互操作
AI智能查询: 智能检索文物信息,提供相关资料和推荐
借展管理: 借展申请、审批、归还的完整流程管理
修复管理: 修复申请、专家分配、进度跟踪
数字档案: 文物档案的创建、管理和导出
消息通知: WebSocket实时消息推送
数据统计: 多维度数据统计和可视化展示
用户管理: 用户和角色的管理,权限控制
系统管理: 操作日志、数据备份等系统功能
系统特色:

前后端分离架构,响应式设计,支持多终端访问
基于角色的权限控制,保障数据安全
丰富的数据可视化功能,支持决策分析
6种主题风格和暗黑模式,个性化体验
完善的操作日志和审计功能

@RestController
@RequestMapping("/api/auth")
@Api(tags = "用户认证")
public class AuthController {
    
    @Autowired
    private SysUserService sysUserService;
    
    @Autowired
    private JwtTokenUtil jwtTokenUtil;
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    /**
     * 用户登录
     */
    @PostMapping("/login")
    @ApiOperation("用户登录")
    public Result<LoginResponse> login(@RequestBody @Valid LoginRequest request) {
        String username = request.getUsername();
        String password = request.getPassword();
        
        // 检查账户是否被锁定
        String lockKey = CacheConstants.LOGIN_LOCKED_KEY + username;
        if (Boolean.TRUE.equals(redisTemplate.hasKey(lockKey))) {
            return Result.error("账户已被锁定,请30分钟后再试");
        }
        
        // 验证用户名和密码
        SysUser user = sysUserService.login(username, password);
        if (user == null) {
            // 记录登录失败次数
            String failKey = CacheConstants.LOGIN_FAILED_KEY + username;
            Integer failCount = (Integer) redisTemplate.opsForValue().get(failKey);
            failCount = failCount == null ? 1 : failCount + 1;
            
            redisTemplate.opsForValue().set(failKey, failCount, 30, TimeUnit.MINUTES);
            
            // 失败5次锁定账户
            if (failCount >= 5) {
                redisTemplate.opsForValue().set(lockKey, "1", 30, TimeUnit.MINUTES);
                return Result.error("登录失败次数过多,账户已被锁定30分钟");
            }
            
            return Result.error("用户名或密码错误,还可尝试" + (5 - failCount) + "次");
        }
        
        // 登录成功,清除失败记录
        redisTemplate.delete(CacheConstants.LOGIN_FAILED_KEY + username);
        
        // 生成Token
        String token = jwtTokenUtil.generateToken(user.getUsername(), user.getRole());
        
        // 构造响应
        LoginResponse response = new LoginResponse();
        response.setToken(token);
        response.setUsername(user.getUsername());
        response.setRealName(user.getRealName());
        response.setRole(user.getRole());
        
        return Result.success(response);
    }
    
    /**
     * 用户注册
     */
    @PostMapping("/register")
    @ApiOperation("用户注册")
    public Result<Void> register(@RequestBody @Valid RegisterRequest request) {
        boolean success = sysUserService.register(request);
        if (success) {
            return Result.success();
        }
        return Result.error("注册失败");
    }
    
    /**
     * 忘记密码
     */
    @PostMapping("/forgot-password")
    @ApiOperation("忘记密码")
    public Result<Void> forgotPassword(@RequestBody @Valid ForgotPasswordRequest request) {
        boolean success = sysUserService.forgotPassword(request);
        if (success) {
            return Result.success();
        }
        return Result.error("密码重置失败");
    }

    }

    public PageResult<CulturalRelic> pageRelics(Integer pageNum, Integer pageSize,

        // 构建查询条件
        QueryWrapper<CulturalRelic> queryWrapper = new QueryWrapper<>();
        
        if (StringUtils.isNotBlank(name)) {
            queryWrapper.like("relic_name", name)
                       .or()
                       .like("relic_code", name);
        }
        
        if (categoryId != null) {
            queryWrapper.eq("category_id", categoryId);
        }
        
        if (StringUtils.isNotBlank(status)) {
            queryWrapper.eq("status", status);
        }
        
        queryWrapper.orderByDesc("create_time");
        
        // 分页查询
        Page<CulturalRelic> page = new Page<>(pageNum, pageSize);
        IPage<CulturalRelic> result = culturalRelicMapper.selectPage(page, queryWrapper);
        
        // 填充分类名称
        for (CulturalRelic relic : result.getRecords()) {
            if (relic.getCategoryId() != null) {
                CulturalRelicCategory category = categoryMapper.selectById(relic.getCategoryId());
                if (category != null) {
                    relic.setCategoryName(category.getCategoryName());
                }
            }
        }
        
        return new PageResult<>(result.getRecords(), result.getTotal());
    }

        // 提取数字部分并加1

        return String.format("WW%05d", num);

A.3 JWT工具类

@Component
public class JwtTokenUtil {
    
    @Value("${jwt.secret}")
    private String secret;
    
    @Value("${jwt.expiration}")
    private Long expiration;
    
    /**
     * 生成Token
     */
    public String generateToken(String username, String role) {
        Map<String, Object> claims = new HashMap<>();
        claims.put("username", username);
        claims.put("role", role);
        
        return Jwts.builder()
                .setClaims(claims)
                .setSubject(username)
                .setIssuedAt(new Date())
                .setExpiration(new Date(System.currentTimeMillis() + expiration * 1000))
                .signWith(SignatureAlgorithm.HS512, secret)
                .compact();
    }
    
    /**
     * 从Token中获取用户名
     */
    public String getUsernameFromToken(String token) {
        try {
            Claims claims = getClaimsFromToken(token);
            return claims.getSubject();
        } catch (Exception e) {
            return null;
        }
    }
    
    /**
     * 从Token中获取角色
     */
    public String getRoleFromToken(String token) {
        try {
            Claims claims = getClaimsFromToken(token);
            return (String) claims.get("role");
        } catch (Exception e) {
            return null;
        }
    }
    
    /**
     * 验证Token是否有效
     */
    public boolean validateToken(String token) {
        try {
            Claims claims = getClaimsFromToken(token);
            Date expiration = claims.getExpiration();
            return expiration.after(new Date());
        } catch (Exception e) {
            return false;
        }
    }
    
    /**
     * 从Token中获取Claims
     */
    private Claims getClaimsFromToken(String token) {
        return Jwts.parser()
                .setSigningKey(secret)
                .parseClaimsJws(token)
                .getBody();
    }
}

// src/api/request.js
import axios from 'axios'
import { ElMessage } from 'element-plus'
import router from '@/router'

// 创建axios实例
const service = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || 'http://localhost:8080',
  timeout: 30000
})

// 请求拦截器
service.interceptors.request.use(
  config => {
    // 添加Token
    const token = localStorage.getItem('token')
    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }
    return config
  },
  error => {
    console.error('请求错误:', error)
    return Promise.reject(error)
  }
)

// 响应拦截器
service.interceptors.response.use(
  response => {
    const res = response.data
    
    // 如果返回的状态码不是200,则认为是错误
    if (res.code !== 200) {
      ElMessage.error(res.message || '请求失败')
      
      // 401: 未授权,跳转到登录页
      if (res.code === 401) {
        localStorage.removeItem('token')
        router.push('/login')
      }
      
      return Promise.reject(new Error(res.message || '请求失败'))
    }
    
    return res
  },
  error => {
    console.error('响应错误:', error)
    
    if (error.response) {
      switch (error.response.status) {
        case 401:
          ElMessage.error('未授权,请重新登录')
          localStorage.removeItem('token')
          router.push('/login')
          break
        case 403:
          ElMessage.error('权限不足')
          break
        case 404:
          ElMessage.error('请求的资源不存在')
          break
        case 500:
          ElMessage.error('服务器错误')
          break
        default:
          ElMessage.error(error.response.data.message || '请求失败')
      }
    } else {
      ElMessage.error('网络错误,请检查网络连接')
    }
    
    return Promise.reject(error)
  }
)

  }),
