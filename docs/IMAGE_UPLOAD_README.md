## 💡 使用建议

### 推荐使用"上传模式"的场景
- 图片在本地电脑上
- 需要长期稳定访问
- 图片是独家资源
- 对图片安全性要求高

### 推荐使用"URL模式"的场景
- 图片已经在网络上
- 图片来自可信的CDN
- 需要节省服务器空间
- 需要频繁更换图片
