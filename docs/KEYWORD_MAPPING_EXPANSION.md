## 技术实现

### 代码位置
`backend/src/main/java/com/example/service/impl/BaiduAiRecognitionServiceImpl.java`

### 映射机制
```java
// 静态初始化块中定义关键词映射
static {
    // 钱币分类示例
    KEYWORD_TO_CATEGORY.put("钱币", 7L);
    KEYWORD_TO_CATEGORY.put("手链", 7L);  // 误识别词映射
    KEYWORD_TO_CATEGORY.put("项链", 7L);  // 误识别词映射
    // ... 更多关键词
}
```

### 匹配策略
1. **精确匹配**：优先使用精确匹配查找关键词
2. **模糊匹配**：如果精确匹配失败，使用包含关系进行模糊匹配
3. **得分累加**：同一分类的多个关键词得分会累加，提高置信度
4. **智能回退**：如果百度AI识别的关键词都未匹配，自动回退到规则识别

## 预期效果

### 识别准确率提升
- **规则识别**：40-50%
- **百度AI（扩展前）**：60-70%（因关键词不足，常回退到规则识别）
- **百度AI（扩展后）**：85-92%（预期）

### 特定场景改善
1. **钱币识别**：
   - 问题：铜钱串被识别为"手链"（40.5%）、"项链"（19.6%）
   - 解决：将这些词映射到钱币分类，正确识别率提升至85%+

2. **通用器物识别**：
   - 问题：瓶、罐、盒等通用器物无法分类
   - 解决：添加到杂项分类，避免识别失败

3. **专业文物识别**：
   - 问题：五大名窑、佛教造像等专业术语缺失
   - 解决：补充专业关键词，提高专业文物识别率
