# 第4章 系统详细设计及实现

本章详细介绍博物馆文物数字化管理系统各个功能模块的设计与实现,包括用户界面设计、程序处理流程设计和核心代码实现。

## 4.1 文物信息管理模块

### 4.1.1 用户界面设计

文物信息管理模块采用表格列表布局,是系统的核心模块之一。界面顶部提供"添加文物"、"批量删除"、"导入"、"导出"等操作按钮,以及搜索筛选条件(按文物名称、分类、状态、年代等)。表格列包括文物编号、文物名称、分类、年代、材质、状态、操作等。每条文物记录提供查看、编辑、删除、生成二维码等操作。点击"添加文物"按钮弹出表单对话框,支持文物基本信息录入和图片上传。文物状态以不同颜色的标签展示(在库为绿色,借出为橙色,修复中为红色)。

### 4.1.2 程序处理流程设计

文物信息管理模块的处理流程如下:

```
开始
  ↓
加载文物列表(分页)
  ↓
用户操作?
  ├─ 添加文物 → 打开表单 → 填写信息 → 上传图片(可选) → 提交 → 生成文物编号 → 刷新列表
  ├─ 编辑文物 → 加载文物详情 → 修改信息 → 提交 → 记录审计日志 → 刷新列表
  ├─ 删除文物 → 确认删除 → 调用删除API → 记录审计日志 → 刷新列表
  ├─ 批量删除 → 选择多条记录 → 确认删除 → 循环调用删除API → 刷新列表
  ├─ 批量修改状态 → 选择多条记录 → 选择新状态 → 提交 → 记录审计日志 → 刷新列表
  ├─ 导入数据 → 选择Excel文件 → 验证数据 → 批量插入 → 返回导入结果
  ├─ 导出数据 → 选择格式(Excel/PDF/Word) → 生成文件 → 下载
  ├─ 生成二维码 → 选择文物 → 生成二维码图片 → 显示/下载
  └─ 搜索筛选 → 输入条件 → 重新加载列表
  ↓
结束
```

## 4.2 3D文物展示模块

3D文物展示模块基于Three.js实现,提供文物的三维可视化展示功能。界面采用全屏沉浸式设计,中央区域为3D模型展示区,支持鼠标拖拽旋转、滚轮缩放、右键平移等交互操作。右侧提供控制面板,包含"自动旋转"、"重置视角"、"全屏显示"等按钮,以及光照强度、背景颜色等调节选项。顶部显示文物基本信息(名称、编号、年代、材质等)。底部提供"返回列表"按钮和模型加载进度条。支持GLTF、GLB、OBJ等多种3D模型格式。

```java
@RestController
@RequestMapping("/relics")
@CrossOrigin
public class Relic3DController {

    @Value("${file.upload.path:uploads/3d-models}")
    private String uploadPath;

    @Value("${file.upload.url-prefix:http://localhost:8080/uploads/3d-models}")
    private String urlPrefix;

    @Autowired
    private CulturalRelicService culturalRelicService;
    
    @Autowired
    private SysOperationLogService operationLogService;
    
    @Autowired
    private UserContextUtil userContextUtil;

    /**
     * 上传3D模型
     */
    @PostMapping("/{id}/3d-model")
    @PreAuthorize("hasAnyRole('ADMIN', 'CURATOR')")
    public Result<Map<String, Object>> upload3DModel(
            @PathVariable Long id,
            @RequestParam("file") MultipartFile file,
            HttpServletRequest httpRequest) {
        
        if (file.isEmpty()) {
            return Result.error("文件不能为空");
        }

        // 检查文件类型
        String originalFilename = file.getOriginalFilename();
        if (originalFilename == null) {
            return Result.error("文件名不能为空");
        }

        String extension = originalFilename.substring(
            originalFilename.lastIndexOf(".")).toLowerCase();
        if (!extension.equals(".gltf") && !extension.equals(".glb") 
            && !extension.equals(".obj")) {
            return Result.error("只支持 GLTF (.gltf, .glb) 和 OBJ (.obj) 格式");
        }

        // 检查文件大小(50MB)
        if (file.getSize() > 50 * 1024 * 1024) {
            return Result.error("文件大小不能超过 50MB");
        }

        try {
            // 获取操作前的文物数据
            CulturalRelic oldRelic = culturalRelicService.getById(id);
            if (oldRelic == null) {
                return Result.error("文物不存在");
            }
            oldRelic = cloneRelic(oldRelic);
            
            // 创建上传目录
            File uploadDir = new File(uploadPath);
            if (!uploadDir.exists()) {
                uploadDir.mkdirs();
            }

            // 生成唯一文件名
            String filename = UUID.randomUUID().toString() + extension;
            Path filePath = Paths.get(uploadPath, filename);

            // 保存文件
            Files.copy(file.getInputStream(), filePath);

            // 构建访问URL
            String modelUrl = urlPrefix + "/" + filename;

            // 更新数据库中的文物记录
            CulturalRelic relic = culturalRelicService.getById(id);
            relic.setModel3dUrl(modelUrl);
            relic.setModel3dUploadTime(LocalDateTime.now());
            culturalRelicService.updateById(relic);

            // 记录审计日志
            try {
                CulturalRelic newRelic = culturalRelicService.getById(id);
                String realName = userContextUtil.getCurrentUserRealName();
                Long userId = userContextUtil.getCurrentUserId();
                String ipAddress = getClientIp(httpRequest);
                
                operationLogService.logDataChange(
                    userId, realName, "上传3D模型", "3D模型管理", "RELIC",
                    id, oldRelic, newRelic, ipAddress, 
                    "POST", "/relics/" + id + "/3d-model"
                );
            } catch (Exception e) {
                System.err.println("记录审计日志失败: " + e.getMessage());
            }

            // 返回结果
            Map<String, Object> result = new HashMap<>();
            result.put("relicId", id);
            result.put("modelUrl", modelUrl);
            result.put("filename", filename);
            result.put("fileSize", file.getSize());

            return Result.success(result);

        } catch (IOException e) {
            return Result.error("文件上传失败: " + e.getMessage());
        }
    }

            ip = request.getHeader("Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("WL-Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getRemoteAddr();
        }
        return ip;
    }
    
    private CulturalRelic cloneRelic(CulturalRelic relic) {
        if (relic == null) return null;
        CulturalRelic clone = new CulturalRelic();
        clone.setId(relic.getId());
        clone.setRelicCode(relic.getRelicCode());
        clone.setRelicName(relic.getRelicName());
        clone.setModel3dUrl(relic.getModel3dUrl());
        clone.setModel3dUploadTime(relic.getModel3dUploadTime());
        return clone;
    }
```

### 4.3.2 程序处理流程设计

AI对话模块的处理流程如下:

```

  ↓
加载用户会话列表
  ↓
用户操作?
  ├─ 新建会话 → 创建会话记录 → 设置默认标题 → 刷新会话列表 → 切换到新会话

  ↓

  ↓
结束
```

### 4.5.1 用户界面设计

用户管理模块采用表格列表布局,展示系统中的所有用户信息。界面顶部提供"添加用户"按钮和搜索筛选条件(按真实姓名、角色等)。表格列包括用户名、真实姓名、角色、邮箱、手机号、状态、操作等。用户状态以不同颜色的标签展示(激活为绿色,禁用为灰色)。点击"添加用户"按钮弹出表单对话框,包含用户名、密码、真实姓名、角色选择、邮箱、手机号等字段。如果选择借展人角色,还需要选择关联的博物馆。每条用户记录提供编辑、删除、解锁账户等操作。

### 4.5.2 程序处理流程设计

用户管理模块的处理流程如下:

```
开始
  ↓
加载用户列表(分页)
  ↓
加载角色列表(用于下拉选择)
  ↓
用户操作?
  ├─ 添加用户 → 打开表单 → 填写信息 → 选择角色 → 如果是借展人则选择博物馆 → 提交 → 密码加密 → 保存用户 → 创建博物馆关联(如有) → 刷新列表
  ├─ 编辑用户 → 加载用户详情 → 修改信息 → 提交 → 更新用户 → 更新博物馆关联(如有) → 记录审计日志 → 刷新列表
  ├─ 删除用户 → 确认删除 → 调用删除API → 删除博物馆关联 → 记录审计日志 → 刷新列表
  ├─ 解锁账户 → 确认解锁 → 清除登录失败次数 → 清除锁定状态 → 提示成功
  ├─ 查看个人信息 → 加载当前用户信息 → 显示详情
  ├─ 修改个人信息 → 打开表单 → 修改信息 → 提交 → 更新用户 → 记录审计日志 → 刷新
  └─ 搜索筛选 → 输入条件 → 重新加载列表
  ↓
结束
```

```java
            userMuseum.setUserId(user.getId());
            userMuseum.setMuseumId(museumId);

    /**
     * 更新用户
     */
    @PutMapping
    @Transactional
    public Result<Boolean> update(@RequestBody Map<String, Object> requestData,
                                  HttpServletRequest httpRequest) {
        Long userId = requestData.get("id") != null ? 
            Long.valueOf(requestData.get("id").toString()) : null;
        
        // 获取修改前的数据
        SysUser oldUser = userId != null ? sysUserService.getUserById(userId) : null;
        
        // 执行更新操作
        SysUser user = new SysUser();
        user.setId(userId);
        user.setUsername((String) requestData.get("username"));
        user.setRealName((String) requestData.get("realName"));
        user.setEmail((String) requestData.get("email"));
        user.setPhone((String) requestData.get("phone"));
        user.setRoleId(requestData.get("roleId") != null ? 
            Long.valueOf(requestData.get("roleId").toString()) : null);
        user.setStatus(requestData.get("status") != null ? 
            Integer.valueOf(requestData.get("status").toString()) : 1);
        user.setUpdateTime(LocalDateTime.now());
        
        boolean success = sysUserService.updateById(user);
        
        // 如果提供了博物馆ID,更新关联
        if (success && requestData.get("museumId") != null) {
            Long museumId = Long.valueOf(requestData.get("museumId").toString());
            // 先删除旧关联
            userMuseumMapper.deleteByUserId(user.getId());
            // 创建新关联
            UserMuseum userMuseum = new UserMuseum();
            userMuseum.setUserId(user.getId());
            userMuseum.setMuseumId(museumId);
            userMuseum.setIsPrimary(1);
            userMuseumMapper.insert(userMuseum);
        }
        
        // 记录审计日志
        if (success && oldUser != null) {
            try {
                SysUser newUser = sysUserService.getUserById(userId);
                String realName = userContextUtil.getCurrentUserRealName();
                Long operatorId = userContextUtil.getCurrentUserId();
                String ipAddress = getClientIp(httpRequest);
                
                operationLogService.logDataChange(
                    operatorId, realName, "修改", "用户管理", "USER",
                    userId, oldUser, newUser, ipAddress, "PUT", "/users"
                );
            } catch (Exception e) {
                System.err.println("记录审计日志失败: " + e.getMessage());
            }
        }
        
        return Result.success("更新成功", success);
    }

        }
        
        String currentUsername = authentication.getName();
        SysUser currentUser = sysUserService.getUserByUsername(currentUsername);
        
        if (currentUser == null) {
            return Result.error("用户不存在");
        }
        
        // 保存修改前的数据
        SysUser oldUser = sysUserService.getUserById(currentUser.getId());
        
        // 构建更新对象
        SysUser user = new SysUser();
        user.setId(currentUser.getId());
        
        // 检查用户名是否修改
        String newUsername = (String) requestData.get("username");
        if (newUsername != null && !newUsername.equals(currentUsername)) {
            // 检查新用户名是否已存在
            SysUser existingUser = sysUserService.getUserByUsername(newUsername);
            if (existingUser != null && 
                !existingUser.getId().equals(currentUser.getId())) {
                return Result.error("用户名已存在");
            }
            user.setUsername(newUsername);
        }
        
        // 更新其他字段
        user.setEmail((String) requestData.get("email"));
        user.setPhone((String) requestData.get("phone"));
        
        // 如果提供了密码,则更新密码
        String password = (String) requestData.get("password");
        if (password != null && !password.trim().isEmpty()) {
            user.setPassword(password);
        }
        
        user.setUpdateTime(LocalDateTime.now());
        
        boolean success = sysUserService.updateById(user);
        
        // 如果是借展人且提供了博物馆ID,更新关联
        if (success && "LOANER".equals(currentUser.getRoleCode()) && 
            requestData.get("museumId") != null) {
            Long museumId = Long.valueOf(requestData.get("museumId").toString());
            // 先删除旧关联
            userMuseumMapper.deleteByUserId(user.getId());
            // 创建新关联
            UserMuseum userMuseum = new UserMuseum();
            userMuseum.setUserId(user.getId());
            userMuseum.setMuseumId(museumId);
            userMuseum.setIsPrimary(1);
            userMuseum.setCreateTime(LocalDateTime.now());
            userMuseum.setUpdateTime(LocalDateTime.now());
            userMuseumMapper.insert(userMuseum);
        }
        
        // 记录审计日志
        if (success && oldUser != null) {
            try {
                SysUser newUser = sysUserService.getUserById(currentUser.getId());
                String ipAddress = getClientIp(httpRequest);
                
                operationLogService.logDataChange(
                    currentUser.getId(), currentUsername, "修改", "个人信息", "USER",
                    currentUser.getId(), oldUser, newUser, ipAddress, 
                    "PUT", "/users/profile"
                );
            } catch (Exception e) {
                System.err.println("记录审计日志失败: " + e.getMessage());
            }
        }
        
        return Result.success("个人信息更新成功", success);
    }
    
    private String getClientIp(HttpServletRequest request) {
        String ip = request.getHeader("X-Forwarded-For");
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("WL-Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getRemoteAddr();
        }
        return ip;
    }
```

```java
@RestController
@RequestMapping("/api/repair")
public class RepairRecordController {

    @Autowired
    private RepairRecordService repairRecordService;

    @Autowired
    private CulturalRelicService culturalRelicService;

    @Autowired
    private NotificationService notificationService;

    /**
     * 分页查询修复记录
     */
    @GetMapping("/page")
    public Result<PageResult<RepairRecord>> getRepairPage(
            @RequestParam(defaultValue = "1") Integer pageNum,
            @RequestParam(defaultValue = "10") Integer pageSize,
            @RequestParam(required = false) String keyword,
            @RequestParam(required = false) String status) {

        QueryWrapper<RepairRecord> queryWrapper = new QueryWrapper<>();
        
        if (StringUtils.isNotBlank(keyword)) {
            queryWrapper.like("relic_name", keyword);
        }
        
        if (StringUtils.isNotBlank(status)) {
            queryWrapper.eq("status", status);
        }
        
        queryWrapper.orderByDesc("apply_time");
        IPage<RepairRecord> result = repairRecordService.page(page, queryWrapper);

    /**
     * 申请修复
     */
    @PostMapping("/apply")
    @OperationLog(module = "修复管理", operation = "申请修复")
    public Result<Void> applyRepair(@RequestBody RepairApplyRequest request) {
        // 检查文物状态
        CulturalRelic relic = culturalRelicService.getById(request.getRelicId());
        if ("修复中".equals(relic.getStatus())) {
            return Result.error("该文物正在修复中,无法重复申请");
        }
        
        // 创建修复记录
        RepairRecord record = new RepairRecord();
        record.setRelicId(request.getRelicId());
        record.setRelicName(relic.getRelicName());
        record.setRepairReason(request.getRepairReason());
        record.setDamageDescription(request.getDamageDescription());
        record.setPriority(request.getPriority());
        record.setExpertId(request.getExpertId());
        record.setStatus("待审批");
        record.setApplicantId(SecurityUtils.getUserId());
        record.setApplicantName(SecurityUtils.getUsername());
        record.setApplyTime(LocalDateTime.now());
        
        repairRecordService.save(record);
        
        // 发送通知给审批员
        notificationService.sendToRole("APPROVER", "修复申请", 
            SecurityUtils.getUsername() + "申请修复文物:" + relic.getRelicName());
        
        return Result.success();
    }

    /**
     * 审批修复
     */
    @PostMapping("/{id}/approve")
    @OperationLog(module = "修复管理", operation = "审批修复")
    public Result<Void> approveRepair(
            @PathVariable Long id,
            @RequestBody RepairApproveRequest request) {
        
        RepairRecord record = repairRecordService.getById(id);
        record.setStatus(request.isApproved() ? "待修复" : "已拒绝");
        record.setApproverId(SecurityUtils.getUserId());
        record.setApproverName(SecurityUtils.getUsername());
        record.setApproveTime(LocalDateTime.now());
        record.setApproveRemark(request.getRemark());
        
        repairRecordService.updateById(record);
        
        // 如果审批通过,更新文物状态
        if (request.isApproved()) {
```

```java
@Aspect
@Component
@Slf4j
public class OperationLogAspect {

    @Autowired
    private SysOperationLogService operationLogService;

    @Around("@annotation(operationLog)")
    public Object around(ProceedingJoinPoint joinPoint, OperationLog operationLog) throws Throwable {
        // 记录开始时间
        long startTime = System.currentTimeMillis();
        
        // 获取请求信息
        ServletRequestAttributes attributes = 
            (ServletRequestAttributes) RequestContextHolder.getRequestAttributes();
        HttpServletRequest request = attributes.getRequest();
        
        // 创建日志对象
        SysOperationLog log = new SysOperationLog();
        log.setModule(operationLog.module());
        log.setOperation(operationLog.operation());
        log.setMethod(joinPoint.getSignature().getName());
        log.setRequestMethod(request.getMethod());
        log.setRequestUrl(request.getRequestURI());
        log.setIpAddress(getClientIp(request));
        
        // 获取操作人信息
        try {
            log.setOperatorId(SecurityUtils.getUserId());
            log.setOperatorName(SecurityUtils.getUsername());
        } catch (Exception e) {
            log.setOperatorName("匿名用户");
        }
        
        // 获取请求参数
        Object[] args = joinPoint.getArgs();
        if (args != null && args.length > 0) {
            try {
                log.setRequestParams(JSON.toJSONString(args));
            } catch (Exception e) {
                log.setRequestParams("参数序列化失败");
            }
        }
        
        Object result = null;
        try {
            // 执行目标方法
            result = joinPoint.proceed();
            
            // 记录成功日志
            log.setStatus("SUCCESS");
            log.setErrorMessage(null);
            
        } catch (Exception e) {
            // 记录失败日志
            log.setStatus("FAILURE");
            log.setErrorMessage(e.getMessage());
            throw e;
            
        } finally {
            // 计算执行时间
            long endTime = System.currentTimeMillis();
            log.setExecutionTime(endTime - startTime);
            log.setOperationTime(LocalDateTime.now());
            
            // 异步保存日志
            operationLogService.save(log);
        }
        
        return result;
    }

    /**
     * 获取客户端IP地址
     */
    private String getClientIp(HttpServletRequest request) {
        String ip = request.getHeader("X-Forwarded-For");
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getHeader("WL-Proxy-Client-IP");
        }
        if (ip == null || ip.isEmpty() || "unknown".equalsIgnoreCase(ip)) {
            ip = request.getRemoteAddr();
        }
        return ip;
    }
}
```

```java
        }
    }

    /**
     * 下载备份
     */
    @GetMapping("/download/{id}")
    public void downloadBackup(@PathVariable Long id, HttpServletResponse response) {
        try {
            SysBackup backup = backupService.getById(id);
            File file = new File(backup.getFilePath());
            
            response.setContentType("application/octet-stream");
            response.setHeader("Content-Disposition", 
                "attachment; filename=" + URLEncoder.encode(backup.getFileName(), "UTF-8"));

                byte[] buffer = new byte[1024];

                    os.write(buffer, 0, len);
                }
                os.flush();
```

#### 后端Service实现代码

**文件路径**:`backend/src/main/java/com/example/service/impl/BackupServiceImpl.java`

```java
@Service
@Slf4j
public class BackupServiceImpl extends ServiceImpl<SysBackupMapper, SysBackup> implements BackupService {

    @Value("${spring.datasource.url}")
    private String dbUrl;

    @Value("${spring.datasource.username}")
    private String dbUsername;

    @Value("${spring.datasource.password}")
    private String dbPassword;

    @Value("${backup.path:backend/backups}")
    private String backupPath;

    @Override
    public void createBackup() throws Exception {
        // 创建备份目录
        File backupDir = new File(backupPath);
        if (!backupDir.exists()) {
            backupDir.mkdirs();
        }

        // 生成备份文件名
        String timestamp = LocalDateTime.now().format(DateTimeFormatter.ofPattern("yyyyMMdd_HHmmss"));
        String fileName = "backup_" + timestamp + ".sql";
        String filePath = backupPath + "/" + fileName;

        // 提取数据库名
        String dbName = extractDbName(dbUrl);

        // 执行mysqldump命令
        String command = String.format(
            "mysqldump -u%s -p%s --databases %s --result-file=%s",
            dbUsername, dbPassword, dbName, filePath
        );

        Process process = Runtime.getRuntime().exec(command);
        int exitCode = process.waitFor();

        if (exitCode != 0) {
            throw new RuntimeException("备份失败,退出码: " + exitCode);
        }

        // 保存备份记录
        File file = new File(filePath);
        SysBackup backup = new SysBackup();
        backup.setFileName(fileName);
        backup.setFilePath(filePath);
        backup.setFileSize(file.length());
        backup.setBackupType("MANUAL");
        backup.setIsEncrypted(false);
        backup.setCreateTime(LocalDateTime.now());

        this.save(backup);
    }

    @Override
    public void restoreBackup(Long id) throws Exception {
        SysBackup backup = this.getById(id);
        String dbName = extractDbName(dbUrl);

        // 执行mysql命令恢复数据
        String command = String.format(
            "mysql -u%s -p%s %s < %s",
            dbUsername, dbPassword, dbName, backup.getFilePath()
        );

        Process process = Runtime.getRuntime().exec(new String[]{"sh", "-c", command});
        int exitCode = process.waitFor();

        if (exitCode != 0) {
            throw new RuntimeException("恢复失败,退出码: " + exitCode);
        }
    }

    @Override
    public void deleteBackup(Long id) {
        SysBackup backup = this.getById(id);

        // 删除文件
        File file = new File(backup.getFilePath());
        if (file.exists()) {
            file.delete();
        }

        // 删除记录
        this.removeById(id);
    }

    /**
     * 从JDBC URL中提取数据库名
     */
    private String extractDbName(String url) {
        // jdbc:mysql://localhost:3306/cultural_relics?...
        int lastSlash = url.lastIndexOf('/');
        int questionMark = url.indexOf('?', lastSlash);
        if (questionMark > 0) {
            return url.substring(lastSlash + 1, questionMark);
        } else {
            return url.substring(lastSlash + 1);
        }
    }
}
```
