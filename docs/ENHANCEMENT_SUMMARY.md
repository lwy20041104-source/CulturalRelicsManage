```

    │   └── 消息提示
    ├── 拖拽排序
    │   ├── Sortable初始化
```

## 维护指南

### 添加新快捷键

编辑 `useTableEnhancements.js`:

```javascript
if (e.ctrlKey && e.key === 'x') {
  e.preventDefault()
  // 你的逻辑
  ElMessage.success('快捷键: Ctrl+X - 功能描述')
}
```

同时更新 `ShortcutHelp.vue` 中的快捷键列表。

### 在新页面启用功能

参考 `docs/TABLE_ENHANCEMENTS.md` 中的开发指南。

### 修改拖拽样式

编辑页面的 `<style>` 部分：

```css
:deep(.sortable-ghost) {
  /* 拖拽时的样式 */
}
```

## 已知问题

1. **拖拽排序不持久化**
   - 当前仅在前端生效
   - 需要后端API支持
   - 建议添加"保存排序"按钮

2. **快捷键在输入框中可能冲突**
   - 已处理大部分情况
   - 某些特殊输入框可能需要额外处理

3. **大数据量性能**
   - 建议数据量超过1000行时禁用拖拽
   - 考虑使用虚拟滚动
