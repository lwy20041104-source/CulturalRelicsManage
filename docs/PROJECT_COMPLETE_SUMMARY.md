## 🎉 核心功能模块（27个）

### 一、用户与权限管理（5个模块）

**5. 登录安全增强**

## 🚀 快速开始

### 环境要求
- JDK 8+
- Node.js 16+
- MySQL 8.0
- Redis 6.0+
- Maven 3.6+

### 后端启动
```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 访问地址
- 前端：http://localhost:5173
- 后端：http://localhost:8080

### 默认账号
- 系统管理员：admin / 123456
- 馆长：curator01 / 123456
- 审批员：approver01 / 123456
- 借展人：loaner / 123456

### 重要提示
1. **执行数据库迁移脚本**（3D模型功能）：
   ```bash
   cd backend
   mysql -u root -p cultural_relics < sql/add_3d_model_support.sql
   ```

2. **配置邮件服务**：
   - 修改 `application-mail-qq.yml` 或 `application-mail-163.yml`
   - 在 `application.yml` 中选择邮箱配置

3. **配置短信服务**（可选）：
   - 修改 `application.yml` 中的阿里云SMS配置
   - 或设置 `sms.enabled=false` 禁用
