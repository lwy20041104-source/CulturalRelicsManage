## 核心业务流程架构

### 1. 借展业务流程

```
┌──────────┐      ┌──────────┐      ┌──────────┐      ┌──────────┐
│ 借展申请  │ ───> │ 审批流程  │ ───> │ 借出管理  │ ───> │ 归还管理  │
└──────────┘      └──────────┘      └──────────┘      └──────────┘
     │                  │                  │                  │
     ▼                  ▼                  ▼                  ▼
  填写申请          审批员审批         文物状态更新        归还确认
  选择文物          通过/驳回          借展中              状态恢复
  设置日期          发送通知           逾期检查            完成记录
```

### 2. 修复业务流程

```
┌──────────┐      ┌──────────┐      ┌──────────┐      ┌──────────┐
│ 修复申请  │ ───> │ 审批流程  │ ───> │ 修复进行  │ ───> │ 修复完成  │
└──────────┘      └──────────┘      └──────────┘      └──────────┘
     │                  │                  │                  │
     ▼                  ▼                  ▼                  ▼
  填写原因          保管员审批         分配专家            验收确认
  选择专家          通过/拒绝          进度跟踪            状态恢复
  设置优先级        发送通知           修复中              完成记录
```

### 3. AI查询流程

```
┌──────────┐      ┌──────────┐      ┌──────────┐      ┌──────────┐
│ 用户输入  │ ───> │ 关键词处理│ ───> │ 双层检索  │ ───> │ 结果展示  │
└──────────┘      └──────────┘      └──────────┘      └──────────┘
     │                  │                  │                  │
     ▼                  ▼                  ▼                  ▼
  输入问题          分词处理           馆藏检索            文物列表
  多轮对话          同义词扩展         全网检索            图片展示
  会话管理          关键词匹配         图片抓取            相关推荐
```

### 4. 自动备份流程

```
┌──────────┐      ┌──────────┐      ┌──────────┐      ┌──────────┐
│ 定时检查  │ ───> │ 判断条件  │ ───> │ 执行备份  │ ───> │ 清理过期  │
└──────────┘      └──────────┘      └──────────┘      └──────────┘
     │                  │                  │                  │
     ▼                  ▼                  ▼                  ▼
  每小时检查        检查频率配置       导出数据库          删除旧备份
  读取配置          检查上次时间       压缩文件            保留最新N个
  判断是否执行      判断是否需要       保存记录            释放空间
```
