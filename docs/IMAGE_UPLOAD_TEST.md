## 测试用例

### 测试用例 1：上传图片文件（新增文物）

#### 测试步骤：
1. 登录系统
2. 进入"文物管理"页面
3. 点击"新增文物"按钮
4. 填写必填字段：
   - 文物名称：测试文物001
   - 年代：唐朝
   - 材质：陶瓷
   - 分类：选择任意分类
   - 状态：在库
5. 在"图片"字段，确保选择"上传图片"模式
6. 点击上传区域或拖拽一张图片文件
7. 确认图片预览正常显示
8. 点击"确认"按钮

#### 预期结果：
- ✅ 图片上传成功
- ✅ 文物保存成功
- ✅ 列表中显示新增的文物
- ✅ 文物图片正常显示

#### 验证点：
- 图片文件是否保存到 `uploads/relics/` 目录
- 数据库 `image_path` 字段是否保存了相对路径
- 图片是否可以正常预览和放大

## 常见问题排查

### 问题1：图片上传失败
**排查步骤**：
1. 检查后端服务是否正常运行
2. 检查 `uploads/relics/` 目录是否存在且有写入权限
3. 检查文件大小是否超过限制
4. 查看浏览器控制台和后端日志

### 问题2：图片预览不显示
**排查步骤**：
1. 检查图片URL是否正确
2. 检查网络连接
3. 检查图片代理服务是否正常
4. 查看浏览器控制台错误信息

### 问题3：模式切换异常
**排查步骤**：
1. 清除浏览器缓存
2. 刷新页面重试
3. 检查前端代码是否有错误
4. 查看浏览器控制台
