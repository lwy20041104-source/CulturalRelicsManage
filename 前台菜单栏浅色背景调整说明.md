# 前台菜单栏浅色背景调整说明

## 📅 修改日期
2026年4月24日

---

## 🎯 修改目标

将前台借展人端的顶部菜单栏背景色从深色改为浅色，使其与后台管理端保持一致，并且让通知铃铛、语言切换器等按钮使用默认样式，无需特殊调整。

---

## 🔧 主要修改

### 1. 顶部导航栏背景色

**修改前（深色背景）**：
```css
.portal-header {
  background: linear-gradient(180deg, #8b6f47, #6d5638);
  color: #fdfbf7;
  box-shadow: 0 2px 8px rgba(139, 111, 71, 0.2);
}
```

**修改后（浅色背景）**：
```css
.portal-header {
  background: #ffffff;
  color: #3d2f1f;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
  border-bottom: 1px solid #eadfce;
}
```

---

### 2. 标题和副标题颜色

**修改前（浅色文字）**：
```css
.logo-section h1 {
  color: #fdfbf7;  /* 米白色 */
}

.subtitle {
  color: #f5ede0;  /* 浅米色 */
}
```

**修改后（深色文字）**：
```css
.logo-section h1 {
  color: #3d2f1f;  /* 深棕色 */
}

.subtitle {
  color: #6c5037;  /* 中棕色 */
  opacity: 0.8;
}
```

---

### 3. 用户信息卡片

**修改前（半透明白色）**：
```css
.user-info-below {
  color: #fdfbf7;
  background: rgba(253, 251, 247, 0.15);
  border: 1px solid rgba(253, 251, 247, 0.25);
}
```

**修改后（浅灰色）**：
```css
.user-info-below {
  color: #606266;
  background: #f5f7fa;
  border: 1px solid #eadfce;
}
```

---

### 4. 导航菜单

**修改前（半透明白色）**：
```css
.nav-menu a {
  color: #fdfbf7;
  background: rgba(253, 251, 247, 0.15);
  border: 1px solid rgba(253, 251, 247, 0.2);
}

.nav-menu a.active {
  background: rgba(166, 124, 82, 0.4);
  color: #fdfbf7;
}
```

**修改后（浅灰色）**：
```css
.nav-menu a {
  color: #606266;
  background: #f5f7fa;
  border: 1px solid #eadfce;
}

.nav-menu a:hover {
  background: #e8eaed;
  color: #a67c52;
  border-color: #a67c52;
}

.nav-menu a.active {
  background: #a67c52;
  color: #ffffff;
}
```

---

### 5. 按钮样式

**修改前（适配深色背景）**：
```css
.header-right .el-button {
  border-color: rgba(253, 251, 247, 0.4);
  color: #fdfbf7;
}

.header-right .el-button--danger {
  border-color: rgba(255, 107, 107, 0.5);
  background: rgba(255, 107, 107, 0.15);
  color: #ffb3b3;
}
```

**修改后（标准样式）**：
```css
.header-right .el-button {
  border-color: #dcdfe6;
  color: #606266;
}

.header-right .el-button:hover {
  background: #f5f7fa;
}

.header-right .el-button--danger {
  border-color: #f56c6c;
  color: #f56c6c;
}

.header-right .el-button--danger:hover {
  background: #f56c6c;
  color: #ffffff;
}
```

---

### 6. 删除专用样式覆盖

删除了以下不再需要的样式：
- `.portal-notification` 相关样式
- `.portal-language` 相关样式

**原因**：浅色背景下，通知铃铛和语言切换器使用默认样式即可，无需特殊调整。
