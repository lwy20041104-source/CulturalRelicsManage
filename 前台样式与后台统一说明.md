## 🎯 修改目标

将前台借展人端的消息通知图标、中英文切换按钮、退出登录按钮以及用户姓名的背景色和样式修改为与后台管理端完全一致。

## 🔧 主要修改

### 1. 删除按钮自定义样式

**修改前**：
```css
.header-right .el-button {
  border-color: #dcdfe6;
  color: #606266;
  background: transparent;
}

.header-right .el-button:hover {
  border-color: #c0c4cc;
  background: #f5f7fa;
}

.header-right .el-button--danger {
  border-color: #f56c6c;
  background: transparent;
  color: #f56c6c;
}
```

**修改后**：
```css
/* 删除所有自定义按钮样式，使用Element Plus默认样式 */
```

**原因**：
- 后台没有自定义按钮样式
- 使用Element Plus的默认样式即可
- 保持前后台一致

---

### 2. 修改用户姓名样式

**修改前（有背景卡片）**：
```css
.user-info-below {
  display: flex;
  align-items: center;
  gap: 8px;
  color: #606266;
  background: #f5f7fa;        /* 浅灰色背景 */
  padding: 6px 12px;          /* 有内边距 */
  border-radius: 6px;         /* 有圆角 */
  font-size: 13px;
  border: 1px solid #eadfce;  /* 有边框 */
  margin-top: 8px;
  width: fit-content;
}
```

**修改后（无背景）**：
```css
.user-info-below {
  display: flex;
  align-items: center;
  gap: 8px;
  color: #5b4a38;             /* 与后台一致 */
  background: transparent;     /* 无背景 */
  padding: 0;                 /* 无内边距 */
  border-radius: 0;           /* 无圆角 */
  font-size: 14px;            /* 与后台一致 */
  border: none;               /* 无边框 */
  margin-top: 8px;
  width: fit-content;
}
```

**效果**：
- 与后台的`.welcome`样式一致
- 简洁、清爽
- 无背景卡片

---

### 3. 修改副标题颜色

**修改前**：
```css
.subtitle {
  color: #6c5037;  /* 中棕色 */
}
```

**修改后**：
```css

  color: #9b8d7d;  /* 浅棕色，与后台一致 */
}
```
