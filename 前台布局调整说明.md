## 🔧 修改内容

### 1. HTML结构调整

**文件**：`frontend/src/views/PublicPortalView.vue`

**修改前**：
```vue
<div class="logo-section">
  <h1>🏛️ {{ t('systemTitle') }}</h1>
  <p class="subtitle">{{ t('systemSubtitle') }}</p>
</div>
<div class="header-right">
  <el-button>中文</el-button>
  <el-button>English</el-button>
  <NotificationBell />
  <div class="user-info">
    <el-icon><User /></el-icon>
    <span>{{ userName }}</span>
  </div>
  <el-button type="danger">退出登录</el-button>
</div>
```

**修改后**：
```vue
<div class="logo-section">
  <h1>🏛️ {{ t('systemTitle') }}</h1>
  <p class="subtitle">{{ t('systemSubtitle') }}</p>
  <div class="user-info-below">
    <el-icon><User /></el-icon>
    <span>{{ userName }}</span>
  </div>
</div>
<div class="header-right">
  <!-- 通知铃铛 -->
  <NotificationBell />
  <el-button>中文</el-button>
  <el-button>English</el-button>
  <el-button type="danger">退出登录</el-button>
</div>
```

**关键变化**：
1. 将用户信息从`header-right`移到`logo-section`内
2. 创建新的`user-info-below`类来显示用户信息
3. 调整`header-right`内元素顺序：通知铃铛 → 中文 → English → 退出登录

---

### 2. CSS样式调整

**文件**：`frontend/src/views/PublicPortalView.vue` (style部分)

#### 2.1 header-container样式

**修改前**：
```css
.header-container {
  max-width: 1400px;
  margin: 0 auto;
  padding: 0 20px;
}
```

**修改后**：
```css
.header-container {
  max-width: 1400px;
  margin: 0 auto;
  padding: 0 20px;
  position: relative;  /* 新增：为绝对定位提供参考 */
}
```

---

#### 2.2 header-right样式

**修改前**：
```css
.header-right {
  display: flex;
  align-items: center;
  gap: 16px;
  margin-top: 12px;
}
```

**修改后**：
```css
.header-right {
  position: absolute;  /* 新增：绝对定位 */
  top: 20px;          /* 新增：距离顶部20px */
  right: 20px;        /* 新增：距离右边20px */
  display: flex;
  align-items: center;
  gap: 12px;          /* 调整：间距从16px改为12px */
}
```

**关键变化**：
- 使用绝对定位将整个`header-right`区域固定在右上角
- 调整元素间距为12px，使布局更紧凑

---

#### 2.3 新增user-info-below样式

**新增样式**：
```css
.user-info-below {
  display: flex;
  align-items: center;
  gap: 8px;
  color: #fdfbf7;
  background: rgba(253, 251, 247, 0.15);
  padding: 6px 12px;
  border-radius: 6px;
  font-size: 13px;
  border: 1px solid rgba(253, 251, 247, 0.25);
  margin-top: 8px;
  width: fit-content;
}
```

**样式说明**：
- 与原来的`user-info`样式类似，但尺寸更小
- `margin-top: 8px`：与副标题保持8px间距
- `width: fit-content`：宽度自适应内容
- `font-size: 13px`：字体稍小，更精致
