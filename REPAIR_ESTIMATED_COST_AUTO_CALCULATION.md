```javascript
// 计算材料总费用
const calculateTotalCost = () => {
  return applyForm.materials.reduce((total, material) => {
    return total + (material.totalPrice || 0)
  }, 0)
}
```

## 🎯 功能说明

### 工作流程

1. **选择材料**：
   - 用户从下拉列表中选择修复材料
   - 系统自动填充该材料的单价（从材料库中获取）

2. **输入数量**：
   - 用户输入使用数量
   - 可以修改单价（如果有特殊情况）

3. **添加材料**：
   - 点击"添加材料"按钮
   - 材料被添加到材料列表中
   - 系统自动计算该材料的小计（数量 × 单价）

4. **实时显示总费用**：
   - 材料列表下方实时显示所有材料的总费用
   - 这个总费用就是预估费用
   - 样式：橙色加粗显示，醒目易读

5. **提交申请**：
   - 点击提交按钮
   - 系统自动将计算出的总费用设置为 `estimatedCost`
   - 提交到后端保存

### 计算公式

```
预估费用 = Σ(材料数量 × 材料单价)
```

即：所有材料的小计之和
